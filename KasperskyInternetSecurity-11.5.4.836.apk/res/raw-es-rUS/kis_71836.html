Acerca de Antirrobo

Antirrobo protege sus datos contra el acceso no autorizado y ayuda a localizar el dispositivo en caso de robo o extravío.

Antirrobo le permite realizar las siguientes operaciones de manera remota:

  • Bloquear el dispositivo y determinar su ubicación
  • Activar una alarma audible en el dispositivo
  • Eliminar datos del dispositivo
  • Tomar una foto de la persona que usa actualmente su dispositivo

Además, si la tarjeta SIM se reemplaza en el dispositivo o el dispositivo se activa sin ella, puede bloquear de manera remota el dispositivo y obtener el nuevo número de teléfono. Esto le permite iniciar otras funciones de Antirrobo en el dispositivo perdido.

SIM Watch está disponible solamente en dispositivos con una tarjeta SIM insertada.

Puede iniciar las funciones Antirrobo de manera remota desde cualquier dispositivo a través del sitio web o con comandos SMS especiales. Puede enviar un mensaje SMS estándar con un código especial desde otro dispositivo móvil. Si Kaspersky Internet Security está instalada en el otro dispositivo, puede enviar un comando SMS desde la aplicación. Para enviar un comando SMS, debe saber el código secreto de la aplicación Kaspersky Internet Security instalada en su dispositivo.

Puede permitir o bloquear el inicio remoto de las funciones Antirrobo en el dispositivo. Si el inicio de Antirrobo está bloqueado, no es posible iniciarlo de manera remota.

El acceso a la configuración de Antirrobo está protegido con un código secreto (especificado durante la configuración inicial de Antirrobo o cuando se inicia Protección de privacidad por primera vez).

Ingreso de llamada o mensaje de texto

Filtro de llamadas y SMS está disponible únicamente en dispositivos con una tarjeta SIM insertada.

La aplicación le solicita que seleccione la acción que se debe realizar al recibir un mensaje SMS entrante o al completar las llamadas entrantes.

La aplicación le solicita que seleccione una acción si se cumplen todas estas condiciones: El modo Estándar de Filtro de llamadas y SMS está establecido y el número no está en las listas de contactos bloqueados o permitidos.

Puede elegir entre una de las siguientes acciones:

  • Bloquear siempre.

    La aplicación marca las llamadas (o los mensajes SMS) del contacto como no deseadas. La aplicación agrega el número del contacto del que se recibió la llamada (o el mensaje SMS) a la lista de contactos bloqueados.

    Si se recibe una llamada del contacto, la aplicación bloquea todas las llamadas subsiguientes entrantes de este contacto. Pero si se recibe un mensaje SMS del contacto, la aplicación le solicita que seleccione una acción para realizar con el mensaje entrante.

    Si se recibe un mensaje SMS del contacto, la aplicación lo bloquea, al igual que todos los mensaje SMS subsiguientes entrantes de este contacto. Pero si se recibe una llamada del contacto, la aplicación le solicita que seleccione una acción para realizar con la llamada entrante.

  • Permitir siempre.

    La aplicación marca las llamadas (o los mensajes SMS) del contacto como deseadas. La aplicación agrega el número del contacto del que se recibió la llamada (o el mensaje SMS) a la lista de contactos permitidos.

    Si se recibe una llamada del contacto, la aplicación permite todas las llamadas subsiguientes entrantes de este contacto. Pero si se recibe un mensaje SMS del contacto, la aplicación le solicita que seleccione una acción para realizar con el mensaje entrante.

    Si se recibe un mensaje SMS del contacto, la aplicación lo permite, al igual que todos los mensaje SMS subsiguientes entrantes de este contacto. Pero si se recibe una llamada del contacto, la aplicación le solicita que seleccione una acción para realizar con la llamada entrante.

  • Omitir.

    La aplicación no adopta ninguna medida sobre la llamada (o SMS) de este contacto. La aplicación no agrega el número del contacto del que se recibió la llamada (o el mensaje SMS) a ninguna de las listas.

    Cuando se recibe la siguiente llamada entrante (o mensaje SMS) de este contacto, la aplicación le solicita que seleccione una acción para realizar con el mensaje entrante.

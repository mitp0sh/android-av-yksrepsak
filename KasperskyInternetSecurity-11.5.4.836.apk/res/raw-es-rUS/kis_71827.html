Acerca de Protección web y Antiphishing de texto

Con Protección web y Antiphishing de texto, puede visitar sitios web confiables y utilizar datos personales en la red de modo seguro.

La protección web y Antiphishing de texto están disponibles únicamente para la versión premium de Kaspersky Internet Security.

Antiphishing de texto solo está disponible en dispositivos con una tarjeta SIM insertada.

Protección web bloquea sitios web malintencionados, que tienen por objetivo distribuir códigos malintencionados, y sitios web fraudulentos (de suplantación de identidad), diseñados para robar sus datos confidenciales y obtener acceso a sus cuentas financieras.

El texto de Antiphishing bloquea vínculos de SMS a sitios web maliciosos o fraudulentos.

Protección web comprueba los sitios web antes de abrirlos. Antiphishing de texto también comprueba vínculos en mensajes SMS antes de abrirlos. Protección web y Antiphishing de texto realizan comprobaciones mediante el uso del servicio en nube Kaspersky Security Network (un servicio en línea dedicado de Kaspersky Lab que contiene información acerca de la confiabilidad de los archivos, las aplicaciones y los recursos de Internet).

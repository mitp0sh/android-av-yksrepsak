Comprobación de aplicaciones del sistema

Puede configurar la aplicación para que omita el análisis de las aplicaciones del sistema instaladas previamente por el fabricante de su dispositivo o que forman parte del firmware.

Si se selecciona esta casilla, la aplicación no analizará las aplicaciones del sistema en la carpeta system/app.

Si no se selecciona esta casilla, la aplicación analizará las aplicaciones del sistema en la carpeta system/app.

Esta casilla está seleccionada de manera predeterminada.

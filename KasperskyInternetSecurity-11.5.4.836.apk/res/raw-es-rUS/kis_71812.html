Si falla la desinfección

La acción que realiza la aplicación en el archivo detectado si falla la desinfección o si no se activa la casilla Desinfectar. Puede elegir entre una de las siguientes acciones:

  • Cuarentena: la aplicación mueve el archivo detectado a Cuarentena.
  • Alertar en caso de detección: la aplicación le solicita que seleccione la acción que se debe realizar en el archivo detectado.
  • Eliminar: la aplicación elimina el archivo detectado.
  • Omitir: la aplicación no realiza ninguna acción en el archivo detectado y no lo elimina del dispositivo.

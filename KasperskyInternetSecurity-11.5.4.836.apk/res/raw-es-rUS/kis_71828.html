Navegadores compatibles

Protección web comprueba sitios web únicamente en el navegador estándar de Android; no los comprueba en ningún otro navegador.

Cuando está habilitada, Protección web comprueba automáticamente cuál es el navegador predeterminado. Si Protección web no es compatible con el navegador predeterminado actual, le solicitará que lo cambie al navegador estándar de Android.

Para usar Protección web en todo momento durante la navegación en Internet, establezca el navegador estándar de Android como el navegador predeterminado.

Si no desea cambiar el navegador predeterminado actual, inicie al navegador estándar de Android para usar datos personales en Internet de manera segura.

Puede iniciar el navegador estándar de Android desde Kaspersky Internet Security o desde el menú de aplicaciones del dispositivo. Si Protección web está habilitada, bloquea los sitios web fraudulentos y malintencionados antes de que se abran.

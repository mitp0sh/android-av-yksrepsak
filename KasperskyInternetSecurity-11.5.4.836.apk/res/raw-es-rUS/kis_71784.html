Configuración de la protección

La configuración de la protección puede efectuarse solamente para la versión premium de Kaspersky Internet Security.

Para configurar la protección:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Antivirus.
  2. Seleccione un valor para la opción Modo de protección Recomendado o Extendido.
  3. Habilite el análisis adicional con la ayuda del servicio en línea de Kaspersky Security Network (KSN). Para hacerlo, seleccione la casilla Cloud Security Scan.
  4. Habilite el análisis de adware, marcadores y aplicaciones de administración remota. Para hacerlo, seleccione la casilla Adware, marcadores y otros.

    Este tipo de análisis detecta aplicaciones que pueden explotarse por intrusos para perjudicar a usuarios.

  5. Si ha seleccionado el modo de protección extendida, presione Protección.

    Si ha seleccionado el modo de protección recomendado, la sección Protección no está disponible.

  6. Seleccione los tipos de archivos para analizar en tiempo real. Para ello, seleccione un valor para la opción Tipo de archivos para analizar.
  7. Seleccione la acción que realizará la aplicación al detectar una amenaza durante el análisis en tiempo real. Para ello, seleccione un valor para la configuración Acción al detectar una amenaza.

Recuperación de código secreto

Para recuperar el código secreto de la aplicación de manera segura:

  1. Abra Web Management en cualquier dispositivo.
  2. Inicie sesión en Web Management con su Cuenta de Kaspersky que usó durante la configuración inicial de la aplicación.
  3. Seleccione el dispositivo del cual necesita recuperar el código secreto.
  4. Haga clic en el botón Recuperación de código secreto.

    El código de recuperación se muestra en Web Management.

  5. Inicie Kaspersky Internet Security en el dispositivo.
  6. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Antirrobo. 
  7. Cuando se le solicite que escriba el código secreto, presione Menú > Recuperación de código secreto.
  8. Ingrese el código de recuperación que se muestra en Web Management.

    El código secreto aparece en la pantalla del dispositivo.

  9. Escriba el código secreto en el campo Escribir el código secreto restaurado.

La aplicación abrirá la ventana Configuración de Antirrobo.

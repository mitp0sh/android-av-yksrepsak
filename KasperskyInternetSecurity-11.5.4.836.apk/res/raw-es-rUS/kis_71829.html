Análisis constante de sitios web

Para habilitar el análisis constante de los sitios web:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Protección web.
  2. Seleccione un valor para la opción Modo de protección Estándar o Extendido.
  3. Presione Cambiar navegador (este botón se muestra si Protección web está habilitado y el navegador estándar de Android no está configurado como navegador predeterminado).

    Se inicia el asistente de selección de navegador predeterminado.

  4. Siga las instrucciones del asistente.

    Cuando finaliza el asistente, el navegador estándar de Android se establece como navegador predeterminado.

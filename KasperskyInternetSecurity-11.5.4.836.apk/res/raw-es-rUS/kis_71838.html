Objetivo del código secreto

Se solicita el código secreto en las siguientes situaciones:

  • Para acceder a la configuración de Antirrobo y Protección de privacidad.
  • Al enviar un comando SMS desde otro dispositivo móvil para activar de manera remota la alarma del dispositivo, bloquear el dispositivo, localizarlo en el mapa, eliminar datos u ocultar contactos confidenciales e información relacionada.

Se le solicitará que especifique el código secreto para la aplicación durante la configuración inicial de Antirrobo o al ejecutar por primera vez Protección de privacidad. Puede cambiar más adelante el código secreto instalado.

El código secreto consta de numerales. La cantidad mínima de caracteres para el código secreto es cuatro.

Si olvida el código secreto, puede restaurarlo.

Actualización programada

Puede configurar las actualizaciones automáticas de la versión de la aplicación y las bases de datos antivirus (en adelante "actualización"). Para llevarlo a cabo, puede crear una programación para el inicio: elija la frecuencia además del día y la hora del inicio si fuera necesario. La aplicación actualizará automáticamente sus bases de datos antivirus y la versión de acuerdo con la programación.

Las actualizaciones automáticas están disponibles únicamente para la versión premium de Kaspersky Internet Security.

Para las actualizaciones, se necesita conexión a Internet.

Para actualizar automáticamente a una hora elegida, el dispositivo debe estar encendido a esa hora.

Para configurar el inicio automáticamente de una actualización programada:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Antivirus > Actualizar.
  2. Configure la frecuencia de las actualizaciones. Para hacer esto, seleccione un valor para la configuración de Programación.
  3. Seleccione un día y hora para el inicio de las actualizaciones. Para ello, seleccione los valores para los parámetros Día de inicio y Hora de inicio.

    Las actualizaciones se iniciarán según el programa.

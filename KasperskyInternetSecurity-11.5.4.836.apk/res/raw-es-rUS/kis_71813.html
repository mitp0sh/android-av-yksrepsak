Inicio de análisis programado

Puede configurar una programación para iniciar automáticamente un análisis completo del dispositivo: elija la frecuencia con que se debe iniciar el análisis, así como el día y la hora del inicio (si es necesario).

Para que la aplicación inicie el análisis automáticamente, el dispositivo se debe activar en ese momento cuando el análisis está programado para comenzar.

La aplicación inicia el análisis programado automático en segundo plano. Cuando se detecta un objeto malintencionado, la aplicación ejecuta la acción seleccionada en la configuración del análisis.

Programación

La frecuencia con la que se realiza un análisis completo del sistema de archivos. Las siguientes opciones están disponibles:

  • Semanalmente: la aplicación inicia automáticamente el análisis una vez por semana. Para esta opción, puede elegir el día y la hora del inicio del análisis.
  • Diariamente: la aplicación inicia automáticamente el análisis todos los días a la hora especificada. Para esta opción, puede elegir la hora del inicio del análisis.
  • Deshabilitado: la aplicación no ejecuta un análisis automático programado. Esto le permite iniciar un análisis en cualquier momento.

 

Día de inicio

Día del análisis. Esta opción de configuración está disponible si se seleccionó Semanalmente para el parámetro Programación.

Hora de inicio

Hora del análisis. Esta opción de configuración está disponible si se seleccionó Diariamente o Semanalmente como valor de Programación.

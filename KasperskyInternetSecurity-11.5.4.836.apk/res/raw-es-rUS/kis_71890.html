Cambiar código secreto

El código secreto se establece cuando la aplicación se ejecuta por primera vez, y se lo usa para hacer lo siguiente:

  • Evitar el acceso no autorizado a la configuración de la aplicación.
  • Habilitar el inicio remoto de las funciones de Antirrobo y Protección de privacidad desde otro dispositivo.

Puede cambiar el código secreto.

Para cambiar el código secreto:

  1. En la ventana principal de Kaspersky Internet Security, presione Configuración > Configuración adicional > Cambiar código secreto.
  2. Escriba el código secreto actual en el campo Escriba el código secreto actual y presione Sgte.
  3. Escriba el nuevo código secreto en el campo Crear un código secreto nuevo y presione Sgte.
  4. Vuelva a escribir el mismo código en el campo Volver a escribir el código nuevo y presione Intro.

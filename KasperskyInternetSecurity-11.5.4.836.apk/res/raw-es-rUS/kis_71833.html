Antiphishing de texto

Antiphishing de texto solo está disponible en dispositivos con una tarjeta SIM insertada.

Esta casilla habilita/deshabilita el análisis constante de enlaces en mensajes de texto entrantes.

Si se selecciona la casilla, Antiphishing de texto analiza enlaces en mensajes de texto entrantes con la ayuda del servicio de nube de Kaspersky Security Network que incluye información sobre la confiabilidad de archivos, aplicaciones y recursos web. Según los resultados del análisis, Antiphishing de texto permite un mensaje de texto que incluye un enlace a un sitio web seguro o solicita al usuario seleccionar una acción en un mensaje de texto que incluye un enlace a un sitio web malicioso o falso.

Si no se selecciona la casilla, Antiphishing de texto no analiza enlaces en mensajes de texto entrantes.

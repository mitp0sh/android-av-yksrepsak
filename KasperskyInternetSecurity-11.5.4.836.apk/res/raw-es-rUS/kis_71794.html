Adware, marcadores automáticos y otros

Detección de adware y otras aplicaciones que los delincuentes pueden usar para dañar el dispositivo o los datos personales.

Las siguientes aplicaciones pertenecen a este tipo de software:

  • Marcadores automáticos
  • Aplicaciones que admiten mensajes SMS pagos salientes
  • Aplicaciones para la administración remota del dispositivo
  • Aplicaciones para controlar la actividad del usuario en el dispositivo (por ejemplo, aplicaciones que solicitan acceso a sus datos personales, enviar mensajes de texto o determinar su ubicación)

Si la casilla se selecciona, Kaspersky Internet Security detecta aplicaciones que pertenecen a estas categorías y le solicitan la acción que se llevará a cabo en ellas. Si no se selecciona la casilla, Kaspersky Internet Security omite las aplicaciones que pertenecen a estas categorías.

Principales funciones de la aplicación

Kaspersky Internet Security proporciona las siguientes funciones principales:

Protección contra virus y otro software malicioso

El dispositivo está protegido contra virus y otro software malicioso mediante el componente Escáner (Antivirus).

El componente se llama Escáner en la versión gratuita y Antivirus en la versión premium.

El análisis solo le permite examinar todo el dispositivo, las aplicaciones instaladas, o una carpeta específica en busca de amenazas, configurar análisis programados del dispositivo, así como actualizar las bases de datos antivirus para obtener una protección actualizada de su información.

Antivirus combina todas las funciones de Escáner con las siguientes funciones adicionales:

  • Protege su dispositivo en tiempo real;
  • Analiza las aplicaciones recién instaladas antes de iniciarlas por primera vez, mediante bases de datos antivirus y el servicio en nube en línea Kaspersky Security Network.
  • Admite las actualizaciones automáticas de las bases de datos antivirus.

Ocultamiento de contactos confidenciales e historial de llamadas y correspondencia SMS relacionados

El componente Protección de privacidad sirve para ocultar contactos confidenciales e información relacionada.

Protección de privacidad está disponible únicamente con la versión premium de Kaspersky Internet Security en dispositivos con una tarjeta SIM insertada.

La Protección de privacidad le permite ocultar temporalmente los contactos confidenciales, al igual que todo su historial de llamadas y los mensajes de SMS. Puede ocultar los contactos e información relacionada usando la configuración de la aplicación o de manera remota con un comando SMS especial.

Protección de datos en el caso de pérdida o robo del dispositivo

Antirrobo protege sus datos contra el acceso no autorizado y ayuda a localizar su dispositivo en caso de robo o extravío.

La función Antirrobo le permite activar de manera remota la alarma del dispositivo, bloquear el dispositivo, determinar la ubicación, eliminar los datos del dispositivo y tomar una foto de la persona que actualmente utiliza el dispositivo. La función Antirrobo le permite iniciar de manera remota las funciones en el dispositivo mediante comandos SMS especiales o mediante Web Management. Para dispositivos con una ranura de tarjeta SIM, cuando se reemplaza la tarjeta SIM o el dispositivo se prende sin una tarjeta SIM, Antirrobo le puede enviar el nuevo número de teléfono del dispositivo en un mensaje SMS o de correo electrónico, o bloquear el dispositivo.

Bloqueo de llamadas y mensajes de texto no deseados

Filtro de llamadas y SMS se usa para bloquear llamadas y mensajes de texto no deseados.

El componente Filtro de llamadas y SMS está disponible solamente en dispositivos con una tarjeta SIM insertada.

Protección contra amenazas en línea

Los componentes Protección web y Antiphishing de texto ofrecen protección contra amenazas en línea.

Los componentes Protección web y Antiphishing de texto están disponibles únicamente para la versión premium de Kaspersky Internet Security.

 

El componente Antiphishing de texto está disponible solamente en dispositivos con una tarjeta SIM insertada.

Protección web bloquea sitios web malintencionados, que tienen por objetivo distribuir códigos malintencionados, y sitios web fraudulentos (de suplantación de identidad), diseñados para robar sus datos confidenciales y obtener acceso a sus cuentas financieras.

El Antiphishing de texto bloquea vínculos de SMS a sitios web maliciosos o falsos.

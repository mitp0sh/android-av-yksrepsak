Envío de comandos SMS desde Kaspersky Internet Security

En caso de robo o pérdida del dispositivo, puede iniciar de manera remota las funciones de Antirrobo y Protección de privacidad en el dispositivo. Para ello, envíe comandos especiales al dispositivo a través de Web Management o de SMS.

Puede enviar un mensaje SMS estándar con un código especial desde otro dispositivo móvil. Si Kaspersky Internet Security está instalada en el otro dispositivo, puede enviar un comando SMS desde la aplicación. Para enviar un comando SMS, debe saber el código secreto de la aplicación Kaspersky Internet Security instalada en su dispositivo.

Los comandos SMS salientes se facturan de acuerdo con el plan de precio del otro dispositivo móvil.

Para iniciar de manera remota las funciones Antirrobo y Protección de privacidad desde otro dispositivo móvil con la aplicación Kaspersky Internet Security instalada:

  1. Abra Kaspersky Internet Security en el otro dispositivo.
  2. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Antirrobo.
  3. Escriba el código secreto.

    Si olvida el código secreto, puede recuperarlo.

  4. Presione Enviar comandos SMS.
  5. Presione Enviar comando SMS ahora, forme el comando y envíelo a su dispositivo. Cuando se crea el comando SMS, utilice el código secreto de la aplicación Kaspersky Internet Security instalada en su dispositivo.

Recibirá un informe de ejecución del comando en el mensaje de SMS de retorno.

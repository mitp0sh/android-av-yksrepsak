Inicio de las funciones de Antirrobo utilizando comandos SMS

Puede iniciar las funciones de Antirrobo en su dispositivo si usa comandos SMS en el caso de insertarse una tarjeta SIM en su dispositivo.

Si está habilitado SIM Watch en el dispositivo perdido, obtendrá el nuevo número de teléfono cuando se reemplace su tarjeta SIM en el dispositivo. El número se le enviará al número de teléfono adicional o a la dirección de correo electrónico especificada en la configuración de Antirrobo. Utilice el nuevo número de teléfono para ejecutar otras funciones de Antirrobo en su dispositivo.

Para ejecutar funciones de Antirrobo utilizando un comando SMS desde un dispositivo diferente, realice una de las siguientes operaciones:

  • Cree un comando en otro dispositivo y envíelo al dispositivo desde Kaspersky Internet Security utilizando la función comando SMS si la aplicación se instala en otro dispositivo.

    Cuando se crea el comando SMS, utilice el código secreto de la aplicación Kaspersky Internet Security instalada en su dispositivo.

  • Envíe un mensaje SMS a su dispositivo con el siguiente texto especial:
    • alarm: <código>: código para activar la alarma y bloquear el dispositivo (donde <código> es el código secreto para Kaspersky Internet Security en su dispositivo);
    • find: <código> – para bloquear el dispositivo y obtener las coordenadas;
    • wipe: <código> – para eliminar los datos personales y los de la tarjeta de memoria;
    • fullreset: <código>: para eliminar todos los datos del dispositivo y revertir su configuración a los valores de fábrica

    Después de haber eliminado todos los datos y de haber restablecido el dispositivo a la configuración de fábrica, ya no podrá recibir ni ejecutar comandos remotos.

Los comandos SMS salientes se facturan de acuerdo con el plan de precio del otro dispositivo móvil.

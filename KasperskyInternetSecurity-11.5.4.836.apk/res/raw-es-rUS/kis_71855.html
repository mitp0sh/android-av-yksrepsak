Envío de comando SMS

Puede ejecutar las funciones de Antirrobo de manera remota u ocultar información confidencial desde cualquier dispositivo que tenga Kaspersky Internet Security. Para hacerlo, debe enviar un comando SMS especial.

Un comando SMS se envía como un SMS cifrado y contiene el código secreto de la aplicación establecido en el otro dispositivo. No se notará la recepción del comando SMS.

La aplicación en ese dispositivo ejecuta el comando SMS si se cumplen estos criterios:

  • el dispositivo tiene señal de red móvil;
  • Kaspersky Internet Security se estableció como administrador de dispositivos;
  • se permite el inicio remoto de esta función en el dispositivo.

Los comandos SMS salientes se facturan a la tarifa actual del proveedor del servicio móvil.

El número de teléfono que recibe el comando SMS

El número de teléfono del dispositivo al que se envía el comando SMS.

Código secreto de la aplicación del dispositivo que recibe el comando SMS

El código secreto de Kaspersky Internet Security del dispositivo que recibe el comando SMS.

Configuración de análisis

Para configurar el análisis:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Antivirus > Analizar.
  2. Seleccione un valor para la configuración Tipo de archivos para analizar.
  3. Seleccione la acción que realizará la aplicación al detectar una amenaza durante el análisis:
    1. Seleccione la casilla Desinfectar si desea que la aplicación intente automáticamente la desinfección al detectar una amenaza.
    2. Para seleccionar la acción que se llevará a cabo sobre una amenaza que no pudo desinfectarse, seleccione el valor para la configuración Si falla la desinfección.
  4. Configure la frecuencia del análisis. Para ello, seleccione Semanalmente o Diariamente para la configuración Programación.
  5. Defina el día y la hora para iniciar el análisis. Para ello, seleccione los valores para los parámetros Día de inicio y Hora de inicio.

    Los análisis se iniciarán de acuerdo con el programa.

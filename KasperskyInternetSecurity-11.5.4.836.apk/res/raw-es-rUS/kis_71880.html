Selección de datos para ocultar

Protección de privacidad está disponible únicamente con la versión premium de Kaspersky Internet Security en dispositivos con una tarjeta SIM insertada.

De forma predeterminada, la aplicación oculta los siguientes datos para contactos confidenciales:

  • entradas del directorio telefónico;
  • mensajes SMS leídos o reenviados y los borradores de los SMS;
  • registros del historial de llamadas.

Si un contacto confidencial ha enviado un mensaje de texto, la aplicación se configura de manera predeterminada para mostrar el mensaje en mensajes de texto entrantes de este contacto sin mostrar el número del contacto. Si un contacto confidencial realiza una llamada, la aplicación permite la llamada entrante sin mostrar el número del contacto.

Puede modificar el tipo de información para ocultar. Por ejemplo, puede habilitar el ocultamiento de mensajes de texto y llamadas entrantes de contactos confidenciales. En este caso, la aplicación bloquea señales de notificación SMS y llamadas entrantes de números confidenciales (en este caso, la parte que llama escucha el tono "ocupado").

Para seleccionar la información para ocultar:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, presione Configuración > Protección de privacidad.
  2. Escriba el código secreto.

    Si olvida el código secreto, puede recuperarlo.

  3. Cambie el botón Protección de privacidad para alternar a APAGADO.
  4. Presione Configuración > Información para ocultar.
  5. Seleccione las casillas junto a aquellos tipos de información que desee ocultar para contactos confidenciales.
  6. Vuelva a la ventana Protección de privacidad y cambie el botón Protección de privacidad para alternar a ACTIVADO.

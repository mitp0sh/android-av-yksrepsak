Cloud Security Scan

Existe un análisis adicional de las aplicaciones recién instaladas que se realiza utilizando el servicio en nube Kaspersky Security Network (un servicio en línea especial de Kaspersky Lab que contiene información acerca de la confiabilidad de los archivos, las aplicaciones y los recursos de Internet). El análisis mediante el servicio en nube le ayuda a proteger sus datos contra amenazas nuevas y desconocidas.

Si se activa la casilla Cloud Security Scan, Kaspersky Internet Security analiza comprobaciones adicionales en las aplicaciones recién instaladas antes de que se inicien por primera vez, a través del servicio de nube Kaspersky Security Network. Si no se activa la casilla, Kaspersky Internet Security no ejecutará análisis adicionales.

Esta opción de configuración está disponible si el Modo de Protección se estableció como Recomendada o Extendida.

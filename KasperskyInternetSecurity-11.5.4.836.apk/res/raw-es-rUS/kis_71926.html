Bloqueo de dispositivo

SIM Watch está disponible si se inserta una tarjeta SIM en el dispositivo. Las siguientes opciones entran en efecto cuando SIM Watch está en ejecución o la función Bloquear y Localizar se inicia por SMS.

Puede habilitar el bloqueo automático si se reemplaza la tarjeta SIM o se activa el dispositivo sin una tarjeta SIM.

Bloquear cuando se reemplaza la tarjeta SIM

Si la casilla está activada, la aplicación bloquea el dispositivo cuando se reemplaza la tarjeta SIM o se activa el dispositivo sin una tarjeta SIM. Para desbloquear el dispositivo, debe escribir el código secreto de la aplicación.

Si la casilla no está activada, la aplicación no bloquea el dispositivo cuando se reemplaza la tarjeta SIM o se activa el dispositivo sin una tarjeta SIM.

Texto cuando se bloquea

El SMS que se muestra en la pantalla cuando se bloquea el dispositivo. De manera predeterminada, se muestra un mensaje estándar.

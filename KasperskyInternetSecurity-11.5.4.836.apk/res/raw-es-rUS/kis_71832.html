Modo de Protección web

Protección web usa el modo para analizar sitios web antes de que se abran. Están disponibles los siguientes modos:

  • Extendido. Protección web analiza todos los sitios web en todas las aplicaciones con conexión a Internet. Protección web bloquea el acceso a Internet de las aplicaciones que se conectan a un sitio web malicioso o falso.
  • Estándar. Protección web bloquea sitios web únicamente en los navegadores admitidos; no los comprueba en otras aplicaciones.
  • Deshabilitado. Protección web no analiza sitios web.

Protección web analiza sitios web antes de que se abran con la ayuda del servicio de nube de Kaspersky Security Network que incluye información sobre la confiabilidad de archivos, aplicaciones y recursos web. Según los resultados del análisis, Protección web abre sitios web de confianza y bloquea sitios web maliciosos diseñados para distribuir código malicioso, y sitios web falsos (phishing) diseñados para robar sus datos confidenciales y obtener acceso a sus cuentas financieras.

Renouveler la licence

Kaspersky Internet Security confirme la présence de la licence d'utilisation de la version premium de l'application chaque fois qu'elle est lancée.

Si vous utilisez la version premium, vous pourrez voir le nombre de jours de validité restants dans les informations relatives à la licence.

Vous pouvez renouveler une licence à partir de 14 jours avant son expiration. Pour ce faire, vous pouvez utiliser une des méthodes suivantes :

  • Acheter une nouvelle licence dans le magasin en ligne.
  • Renouveler la licence sur le Web Management My Kaspersky.

    Le Web Management My Kaspersky est accessible uniquement aux Etats-Unis et en Grande-Bretagne.

  • Utiliser le code d'activation obtenu lors de l'achat de la licence.

Quand la validité de la licence expire, l'application passe automatiquement à la version free.

Pour renouveler la validité de la licence à l'aide du code d'activation :

  1. Dans la fenêtre principale de Kaspersky Internet Security, dans le panneau de lancement rapide, appuyez sur Paramètres > Paramètres avancés > Gestion de la licence > Renouveler la licence.

    L'application peut suggérer d'utiliser une licence détectée pour votre compte dans le Web Management My Kaspersky. Dans ce cas, refusez l'activation à l'aide de la licence détectée en appuyant sur Autres modes d'activation.

    Le Web Management My Kaspersky est accessible uniquement aux Etats-Unis et en Grande-Bretagne.

    Si aucune licence permettant d'activer l'application n'est détectée pour votre compte utilisateur, passez tout de suite à l'étape suivante.

  2. Sélectionnez Saisie du code d'activation.
  3. Saisissez le code d'activation obtenu lors de l'achat de la licence.

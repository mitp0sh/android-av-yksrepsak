Sélection des informations à dissimuler

Le module Protection Vie Privée est accessible uniquement dans la version premium de Kaspersky Internet Security et sur les appareils dotés d'une carte SIM.

Par défaut, l'application masque les informations suivantes pour les contacts confidentiels :

  • les enregistrements dans le répertoire téléphonique ;
  • les SMS reçus lus, les SMS transmis et les brouillons de SMS ;
  • les entrées dans le journal des appels.

Si le contact confidentiel a envoyé un SMS, par défaut l'application affiche le message dans les SMS en provenance de ce contact, mais n'affiche pas le numéro du contact. Si le contact confidentiel a réalisé un appel, l'application autorisé l'appel, mais n'indique pas le numéro du contact.

Vous pouvez modifier le type d'information à dissimuler. Par exemple, vous pouvez activer la dissimulation des appels et des SMS en provenance des contacts confidentiels. Dans ce cas, l'application bloque les signaux de réception de SMS ou d'appels en provenance de numéros confidentiels (dans ce cas, l'auteur de l'appel entend la tonalité "occupé").

Pour sélectionner les informations à dissimuler :

  1. Dans le panneau de lancement rapide de la fenêtre principale de Kaspersky Internet Security, appuyez sur Paramètres > Protection Vie Privée.
  2. Saisissez le code secret de l'application.

    Si vous avez oublié le code secret, vous pouvez le récupérer.

  3. Déplacez le bouton Protection Vie Privée en position désactivé.
  4. Appuyez sur Paramètres > Informations à masquer.
  5. Cochez les cases en regard des types d'information que vous souhaitez dissimuler pour les contacts confidentiels.
  6. Revenez à la fenêtre Protection Vie Privée et déplacez l'interrupteur Protection Vie Privée en position activé.

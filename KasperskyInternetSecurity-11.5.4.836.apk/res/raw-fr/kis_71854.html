Web Management

En cas de perte ou de vol de votre appareil, le Web Management permet de le verrouiller ou de le localiser, d'en supprimer les données ou d'obtenir la photo de la personne qui utilise l'appareil en ce moment. Pour ce faire, ouvrez une session sur le portail à l'aide du compte utilisé pour la configuration initiale de l'application.

Synchronisation

Pendant la synchronisation, l'application reçoit les instructions et renvoie au Web Management les informations relatives à l'exécution de ces instructions.

L'appareil doit être connecté à Internet pour que la synchronisation puisse avoir lieu.

Le délai d'attente pour la synchronisation dépend du type de la licence activée sur l'application.

L'appareil est associé au compte utilisateur

Kaspersky Account permet d'activer, via le Web Management, la fonction Antivol sur votre appareil.

Kaspersky Account est un compte utilisateur unique qui assure l'accès à tous les services de Kaspersky Lab. Si vous êtes abonné à des services tels que l'Espace Personnel ou si vous possédez un compte pour Web Management, cela signifie que vous possédez Kaspersky Account .

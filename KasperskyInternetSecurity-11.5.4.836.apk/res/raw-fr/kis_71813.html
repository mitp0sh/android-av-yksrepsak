Lancement programmé de l'analyse

Vous pouvez programmer l'exécution automatique de l'analyse complète de l'appareil : sélectionner la fréquence d'exécution de l'analyse et l'heure et le jour (le cas échéant).

Pour que l'application puisse lancer l'analyse automatiquement, l'appareil doit être allumé à l'heure programmée.

L'application lance en arrière-plan l'analyse automatique selon la programmation définie. Quand une menace est détectée, l'action définie par le paramètre d'analyse est exécutée sur cet objet.

Programmation

Fréquence selon laquelle l'application va lancer automatiquement l'analyse complète du système de fichiers. Les options suivantes sont prévues :

  • Une fois par semaine : l'application lance l'analyse automatique une fois par semaine. Pour cette option, vous pouvez sélectionner le jour et l'heure de début de l'analyse.
  • Une fois par jour : l'application lance automatiquement l'analyse chaque jour à l'heure indiquée. Pour cette option, vous pouvez sélectionner l'heure de début de l'analyse.
  • Désactivé : l'application ne réalise pas l'analyse programmée. Vous pouvez dans cas lancer l'analyse au moment qui vous convient le mieux.

 

Jour de lancement

Jour de lancement de l'analyse. Le paramètre est accessible si le paramètre Programmation possède la valeur Une fois par semaine.

Heure de début

Heure du lancement de l'analyse. Le paramètre est accessible si le paramètre Programmation possède la valeur Une fois par jour ou Une fois par semaine.

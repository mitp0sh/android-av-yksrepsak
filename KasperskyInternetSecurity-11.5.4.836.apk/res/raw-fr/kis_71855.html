Envoi d'instructions SMS

Vous pouvez lancer la fonction Antivol ou masquer les informations confidentielles sur un autre appareil doté de Kaspersky Internet Security à distance. Il faut pour ce faire envoyer des instructions par SMS.

L'instruction SMS est envoyée sous forme d'un SMS crypté qui contient le code secret de l'application, installée sur l'autre appareil. La réception de l'instruction passera inaperçue sur l'autre appareil.

Sur un autre appareil, l'application exécute l'instruction SMS reçue si les conditions suivantes ont été exécutées :

  • l'appareil est dans une zone de couverture d'un réseau de téléphonie mobile ;
  • l'application Kaspersky Internet Security est installée en tant qu'administrateur de l'appareil ;
  • le lancement à distance de cette fonction est autorisé sur l'appareil.

Le coût du SMS contenant l'instruction envoyé est celui de votre opérateur de téléphonie mobile.

Numéro de téléphone recevant l'instruction SMS

Numéro de téléphone de l'appareil auquel l'instruction SMS est envoyée.

Code secret de l'application sur l'appareil qui reçoit l'instruction SMS

Il s'agit du code secret de Kaspersky Internet Security sur l'appareil qui reçoit l'instruction SMS.

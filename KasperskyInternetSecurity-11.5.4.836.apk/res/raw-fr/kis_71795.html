Type de fichiers analysés

Sélection des types de fichier dans lesquels l'application recherche la présence éventuelle de menaces. L'application analyse les fichiers du type sélectionné dans le cadre de n'importe quelle action (par exemple, ouverture, copie, lancement, déplacement, enregistrement ou modification). Vous avez le choix parmi les types de fichiers suivants :

  • Tous les fichiers : l'application analyse les fichiers de tous les types (par exemple des fichiers musicaux, texte, système ou exécutables).
  • Applications et archives uniquement : l'application analyse uniquement les fichiers exécutables (à savoir les applications ou les distributions d'applications), par exemple, des fichiers au format APK.

    La protection selon ce paramètre fonctionne différemment de la protection selon le mode recommandé. Quand ce paramètre est sélectionné, la protection peut détecter les menaces dans la distribution d'une application (fichier APK), par exemple lors de sa copie sur la carte mémoire. En mode recommandé, Kaspersky Internet Security analyse uniquement les applications installées, mais pas les distributions.

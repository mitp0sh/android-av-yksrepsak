Activation de la version premium de l'application

Afin de pouvoir utiliser toutes les fonctions de l'application, vous devez activer la version premium de Kaspersky Internet Security.

L'activation rend l'application complètement opérationnelle.

L'activation peut être réalisée d'une des manières suivantes :

  • Saisir le code d'activation que vous avez obtenu à l'achat de la licence.
  • Dans l'application, lors de l'activation, utiliser une licence détectée pour votre compte sur dans le Web Management My Kaspersky ou ouvrir le Web Management My Kaspersky et activer l'application à l'aide de celui-ci.

    Le Web Management My Kaspersky est accessible uniquement aux Etats-Unis et en Grande-Bretagne.

  • Acheter une licence dans le magasin en ligne.

Vous pouvez activer la version premium de l'application lors de la première utilisation ou à n'importe quel moment plus tard.

L'activation de l'application requiert une connexion active à Internet.

Pour activer la version premium de l'application à l'aide du code d'activation :

  1. Dans la fenêtre principale de Kaspersky Internet Security, dans le panneau de lancement rapide, appuyez sur Paramètres > Paramètres avancés > Gestion de la licence > Activer la version premium.

    L'application peut suggérer d'utiliser une licence détectée pour votre compte dans le Web Management My Kaspersky. Dans ce cas, refusez l'activation à l'aide de la licence détectée en appuyant sur Autres modes d'activation.

    Le Web Management My Kaspersky est accessible uniquement aux Etats-Unis et en Grande-Bretagne.

  2. Sélectionnez Saisie du code d'activation.
  3. Saisissez le code d'activation dans la fenêtre qui s'ouvre et cliquer sur Activer.

L'application envoie une demande d'activation au serveur d'activation de Kaspersky Lab. Si la demande d'activation réussit, l'application vous le signale et affiche les informations sur la licence.

Pour activer la version premium de l'application sur la base d'une licence détectée pour votre compte utilisateur sur le Web Management My Kaspersky, procédez comme suit :

Ce mode d'activation est disponible uniquement aux Etats-Unis et en Grande-Bretagne.

  1. Dans la fenêtre principale de Kaspersky Internet Security, dans le panneau de lancement rapide, appuyez sur Paramètres > Paramètres avancés > Gestion de la licence > Activer la version premium.

    Si une licence pouvant servir à activer l'application est détectée pour votre compte dans le Web Management My Kaspersky, Kaspersky Internet Security propose de l'utiliser pour l'activation.

    Si aucune licence pouvant servir à activer l'application n'est détectée pour votre compte dans le Web Management My Kaspersky, Kaspersky Internet Security propose d'autres modes d'activation.

  2. Sélectionnez la licence requise, puis appuyez sur Continuer.

L'application envoie une demande d'activation au serveur d'activation de Kaspersky Lab. Si la demande d'activation réussit, l'application vous le signale et affiche les informations sur la licence.

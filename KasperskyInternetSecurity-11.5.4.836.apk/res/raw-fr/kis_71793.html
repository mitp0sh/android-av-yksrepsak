Analyse dans le Cloud

Analyse complémentaire des nouvelles applications avant leur première exécution à l'aide du service Cloud de Kaspersky Security Network (service en ligne de Kaspersky Lab qui contient des informations relatives à la fiabilité des fichiers, des applications et des ressources Internet). L'analyse à l'aide du service Cloud de contribue à la protection des données contre les nouvelles menaces inconnues.

Quand la case Analyse dans le Cloud est cochée, Kaspersky Internet Security soumet les nouvelles applications à une analyse complémentaire avant leur première exécution à l'aide du service dans le nuage Kaspersky Security Network. Si la case n'est pas cochée, Kaspersky Internet Security ne réalise pas l'analyse complémentaire.

Le paramètre est accessible si le paramètre Mode de protection possède la valeur Recommandé ou Avancé.

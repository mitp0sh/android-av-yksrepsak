En cas de détection d'une menace

Si Kaspersky Internet Security détecte une menace dans une application installée, il propose à l'utilisateur de réaliser une opération sur cette application (supprimer ou ignorer).

Si Kaspersky Internet Security détecte la menace non pas dans l'application, mais dans un fichier d'un autre type (par exemple, un fichier texte, un fichier audio, une archive ou une distribution), il place par défaut ce fichier en quarantaine.

Vous pouvez configurer les actions réalisées par défaut suite à la détection d'une menace.

Vous pouvez consulter les informations relatives aux menaces détectées dans les sections Etat et Rapports.

Pour consulter les informations relatives aux menaces détectées, réalisez une des opérations suivantes :

  • Dans la fenêtre principale de Kaspersky Internet Security, appuyez sur le bouclier ou faites glisser l'écran vers la gauche pour accéder à la section Etat.
  • Dans le panneau de lancement rapide de la fenêtre principale de Kaspersky Internet Security, appuyez sur Paramètres > Paramètres avancés > Rapports.

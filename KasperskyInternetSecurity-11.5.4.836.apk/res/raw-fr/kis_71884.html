Masquer via l'instruction SMS

Autorise/interdit la dissimulation de toutes les informations de votre appareil à distance à l'aide d'une instruction SMS hide: <code> (où <code> est le code secret de Kaspersky Internet Security sur votre appareil).

Si la fonction est autorisée sur l'appareil, elle active la dissimulation des données directement après la réception de l'instruction SMS. Si la fonction est interdite sur l'appareil, l'application ne peut exécuter l'instruction de dissimulation des informations envoyée par SMS.

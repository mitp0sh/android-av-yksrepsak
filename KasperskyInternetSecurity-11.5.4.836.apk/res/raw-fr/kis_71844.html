Configuration des actions en cas de remplacement de la carte SIM

Surveillance SIM est disponible uniquement sur les appareils dotés d'une carte SIM.

Après la configuration initiale de l'Antivol, vous pouvez modifier les paramètres de contrôle de manipulation de la carte SIM.

Pour configurer le contrôle des manipulations de la carte SIM :

  1. Dans le panneau de lancement rapide de la fenêtre principale de Kaspersky Internet Security, appuyez sur Paramètres > Antivol. 
  2. Saisissez le code secret de l'application.

    Si vous avez oublié le code secret, vous pouvez le récupérer.

  3. Choisissez Surveillance SIM.
  4. Cochez ou décochez la case Surveillance SIM.
  5. Si la case Surveillance SIM est cochée, dans le groupe Modes de réception du nouveau numéro de téléphone et de l'emplacement de l'appareil, remplissez les champs Numéro de téléphone et Adresse de messagerie afin d'obtenir le nouveau numéro de téléphone par SMS ou courrier électronique en cas de remplacement de la carte SIM.
  6. Le cas échéant, cochez, dans le groupe Verrouillage de l'appareil, la case Verrouiller en cas de remplacement de la carte SIM afin de verrouiller l'appareil en cas de remplacement de la carte SIM ou d'utilisation de l'appareil sans celle-ci.
  7. Le cas échéant, saisissez dans le champ Texte si verrouillage le texte du message qui sera affiché sur l'écran de l'appareil verrouillé.

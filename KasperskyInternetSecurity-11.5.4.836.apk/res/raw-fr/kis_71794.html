Publicité, numéroteur et autres

Détection des logiciels publicitaires et d'autres applications qui peuvent être utilisées par des individus malintentionnés pour nuire à votre appareil et à vos données.

Cette catégorie reprend les applications suivantes :

  • numéroteurs automatiques ;
  • applications permettant d'envoyer des SMS payants ;
  • applications d'administration à distance de l'appareil ;
  • applications permettant d'observer l'activité de l'utilisateur sur l'appareil (par exemple, les applications qui sollicitent l'accès à vos données personnelles, l'envoi des messages ou à la géolocalisation).

Si la case est cochée, Kaspersky Internet Security, détecte les applications de ces catégories et propose une série d'actions à exécuter sur celles-ci. Quand la case est décochée, Kaspersky Internet Security ne détecte pas les applications de ces catégories.

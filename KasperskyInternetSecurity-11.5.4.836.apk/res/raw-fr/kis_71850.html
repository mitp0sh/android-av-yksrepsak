Suppression

Autorisation/interdiction de la suppression à distance des données personnelles uniquement ou de la purge complète de l'appareil l'aide de Web Management ou des instructions SMS wipe: <code> et fullreset: <code> (où <code> est le code secret de Kaspersky Internet Security sur votre appareil).

Le coût de l'envoi du SMS contenant l'instruction est défini par le tarif de l'opérateur de téléphonie mobile de l'appareil distant.

La fonction Suppression permet de supprimer les données suivantes :

  • Uniquement les données personnelles (par exemple, contacts, messages et données relatives au compte utilisateur Google) ainsi que les informations stockées sur la carte mémoire.
  • Toutes les données, y compris les données sur la carte mémoire. Lors de la suppression de toutes les données, la configuration par défaut de l'appareil est restaurée.

Après la suppression de toutes les données et la réinitialisation des paramètres sur leur valeur par défaut, l'appareil n'est plus en mesure de recevoir et d'exécuter les commandes à distance.

Si la fonction est activée sur l'appareil, l'application supprime les données dès qu'elle reçoit l'instruction. Si vous aviez envoyé une instruction par SMS, l'application confirme la suppression des données dans la réponse au SMS.

Le coût du SMS envoyé en réponse est celui pratiqué par votre opérateur de téléphonie mobile.

Si la fonction est interdite sur l'appareil, l'application ne peut exécuter l'instruction de suppression de données sur l'appareil.

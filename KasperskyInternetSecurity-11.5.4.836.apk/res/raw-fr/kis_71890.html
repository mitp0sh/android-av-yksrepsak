Modification du code secret

Le code secret de l'application est défini lors de la première exécution de l'application et ses fonctions sont les suivantes :

  • empêcher l'accès non autorisé aux paramètres de l'application ;
  • lancer à distance depuis un autre appareil la fonction Antivol et la dissimulation des données.

Vous pouvez modifier le code secret.

Pour changer le code secret de l'application, procédez comme suit :

  1. Dans l’écran principal de Kaspersky Internet Security, cliquez sur Paramètres > Paramètres avancés > Modifier le code secret.
  2. Saisir le code secret actuel de l'application dans le champ Saisissez le code secret actuel et appuyez sur Suivant.
  3. Saisir le code secret actuel de l'application dans le champ Nouveau code secret et appuyez sur Suivant.
  4. Saisissez ce code une nouvelle fois dans le champ Ressaisissez le nouveau code et cliquez sur Entrée.

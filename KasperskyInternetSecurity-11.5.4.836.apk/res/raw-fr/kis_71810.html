Réparation des fichiers malveillants

L'application tente de réparer le fichier. Quand la case est cochée, l'application tente d'abord de réparer le fichier détecté. Si l'application ne parvient pas à réparer le fichier détecté, elle exécute l'action définie dans le groupe Si réparation impossible.

Si cette case n'est pas cochée, quand le fichier est détecté, l'application exécute sans délai l'action définie dans le groupe Si réparation impossible.

Type de fichiers analysés

Le choix des fichiers analysés par l'application pendant l'analyse du système de fichiers de l'appareil. Vous pouvez sélectionner les types de fichier suivants :

  • Tous les fichiers : l'application analyse les fichiers de tous les types (par exemple des fichiers musicaux, texte, système ou exécutables). Le paramètre est sélectionné par défaut.
  • Applications et archives uniquement : l'application analyse uniquement les fichiers exécutables (à savoir, les applications et les programmes d'installation du logiciel), par exemple les fichiers au format APK. Les autres fichiers ne sont pas analysés. La vitesse d'analyse de l'appareil augmente avec cette option.

Quel que soit le type de fichiers sélectionnés, l'application analyse le contenu des archives. Par exemple, l'analyse des archives aux formats différents permet d'analyser la carte mémoire utilisée sur plusieurs appareils.

L'application analyse les archives dans les formats pris en charge par Android et d'autres systèmes d'exploitation pour téléphone mobile. L'application permet d'analyser les archives aux formats suivants : ZIP, JAR, JAD, SIS, SISX, CAB et APK.

Quand l'application analyse des archives, elle les décompacte. Par conséquent, l'analyse des archives peut ralentir considérablement l'exécution de l'analyse de l'appareil.

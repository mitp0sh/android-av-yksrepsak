Qu'est-ce que Kaspersky Account

Le code utilisateur est requis pour accéder au Web Management et pour réaliser diverses opérations à distance comme activer l'alarme, le verrouiller, le géolocaliser, supprimer les données qu'il contient ou prendre une photo de la personne qui utilise en ce moment l'appareil à son insu. Le portail permet également de récupérer le code secret de Kaspersky Internet Security si vous l'avez oublié.

Kaspersky Account est un compte utilisateur unique qui assure l'accès à tous les services de Kaspersky Lab. Si vous êtes abonné à des services tels que l'Espace Personnel ou si vous possédez un compte pour Web Management, cela signifie que vous possédez Kaspersky Account .

Vous désignez le compte utilisateur lors de la configuration initiale de l'Antivol. Votre adresse de messagerie fait office de nom de compte utilisateur. Si vous avez oublié votre mot de passe du code utilisateur, vous pouvez le restaurer.

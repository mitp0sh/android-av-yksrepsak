Configuration des paramètres de la protection

La configuration des paramètres de la protection est accessible uniquement dans la version premium de Kaspersky Internet Security.

Pour configurer la protection :

  1. Dans le panneau de lancement rapide de la fenêtre principale de Kaspersky Internet Security, appuyez sur Paramètres > Anti-Virus.
  2. Attribuez la valeur Recommandé ou Avancé au paramètre Mode de protection.
  3. Activez l'analyse complémentaire via le service en ligne Kaspersky Security Network (KSN). Pour ce faire, cochez la case Analyse dans le Cloud.
  4. Activez la recherche d'éventuels logiciels publicitaires, numéroteurs ou applications d'administration à distance de l'appareil. Pour ce faire, cochez la case Publicité, numéroteur et autres.

    Cette analyse permet de détecter les applications que des individus malintentionnés pourraient utiliser pour nuire aux utilisateurs.

  5. Si vous avez sélectionné le mode de protection avancé, appuyez sur Protection.

    Si vous avez sélectionné le mode de protection recommandé, le groupe Protection ne sera pas disponible.

  6. Sélectionnez le type de fichier à soumettre à l'analyse en temps réel. Pour ce faire, attribuez une valeur au paramètre Type de fichiers à analyser.
  7. Sélectionnez l'action qui sera exécutée par l'application en cas de détection d'une menace pendant l'analyse en temps réel. Pour ce faire, attribuez une valeur au paramètre Action si détection de menaces.

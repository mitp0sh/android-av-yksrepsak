Lancement des fonctions de l'Antivol via des instructions SMS

Vous pouvez lancer les fonctions de l'Antivol sur votre appareil à l'aide d'instructions SMS si votre appareil possède une carte SIM.

Si la fonction Surveillance SIM est activée sur l'appareil égaré, vous recevrez le nouveau numéro de téléphone de l'appareil en cas de remplacement de la carte SIM de ce dernier. Ce nouveau numéro sera envoyé au numéro de téléphone ou à l'adresse de messagerie renseignés dans les paramètres de l'Antivol. Utilisez ce nouveau numéro de téléphone pour lancer d'autres fonctions d'Antivol sur votre appareil.

Pour lancer les fonctions de l'Antivol par SMS depuis un autre appareil, réalisez une des opérations suivantes :

  • Depuis un autre appareil, rédigez et envoyez l'instruction à votre appareil depuis Kaspersky Internet Security à l'aide de la fonction d'envoi des instructions SMS (si l'application est installée sur un autre appareil).

    Lors de la rédaction des instructions SMS, utilisez le code secret de Kaspersky Internet Security installé sur votre appareil.

  • Envoyez à votre appareil un SMS contenant le texte suivant :
    • alarm: <code> : déclenchement de l'alarme et verrouillage de l'appareil (<code> représente le code secret de la version de Kaspersky Internet Security installée sur votre appareil) ;
    • find: <code> : verrouillage de l'appareil et géolocalisation ;
    • wipe: <code> : pour supprimer les données personnelles et les informations stockées sur la carte mémoire ;
    • fullreset: <code> : pour supprimer toutes les données stockées sur l'appareil et rétablir la configuration par défaut.

    Après la suppression de toutes les données et la réinitialisation des paramètres sur leur valeur par défaut, l'appareil n'est plus en mesure de recevoir et d'exécuter les commandes à distance.

Le coût de l'envoi du SMS contenant l'instruction est défini par le tarif de l'opérateur de téléphonie mobile de l'appareil distant.

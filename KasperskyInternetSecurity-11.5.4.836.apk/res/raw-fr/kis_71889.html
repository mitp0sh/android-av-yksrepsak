Widget

L'indicateur de couleur et l'image dans le widget logiciel de l'écran principal indiquent l'état de la protection de votre appareil. Ils permettent également d'accéder à la configuration des paramètres de l'application.

Le vert indique que le mode de protection sélectionné est le mode recommandé ou avancé. Le bouclier gris signifie que la protection est désactivée. Le point d'exclamation dans le triangle jaune signifie que la licence a expiré.

Vous pouvez décider d'afficher ou non le widget sur l'écran principal de l'appareil.

Activer le widget

Autorisation/interdiction de l'affichage du widget sur l'écran principal de l'appareil. Si la case est cochée, le widget est affiché dans la liste des widgets disponibles. Si la case n'est pas cochée, après le redémarrage de l'appareil, le widget n'est pas affiché dans la liste des widgets disponibles et il est impossible de l'ajouter à l'écran principal de l'appareil.

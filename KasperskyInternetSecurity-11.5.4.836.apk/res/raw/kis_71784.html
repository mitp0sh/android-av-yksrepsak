Configuring protection settings

Protection settings can be configured only for the premium version of Kaspersky Internet Security.

To configure protection settings:

  1. On the quick launch panel in the main window of Kaspersky Internet Security, tap Settings > Anti-Virus.
  2. Select a value for the Protection mode setting Recommended or Extended.
  3. Enable additional scanning with the help of the Kaspersky Security Network (KSN) online service. To do so, select the Cloud Security Scan check box.
  4. Enable scanning for adware, dialers, and remote administration apps. To do so, select the Adware, auto-dialers, and other check box.

    This type of scan detects apps that can be exploited by intruders to harm users.

  5. If you have selected the extended protection mode, tap Protection.

    If you have selected the recommended protection mode, the Protection section is unavailable.

  6. Select the types of files to scan in real time. Do so by selecting a value for the Type of files to scan setting.
  7. Select the action to be performed by the app on detecting a threat during real-time scanning. Do so by selecting a value for the Action on threat detection setting.

Text Anti-Phishing

The Text Anti-Phishing is available only on devices with an inserted SIM card.

This check box enables / disables constant scanning of links in incoming text messages.

If the check box is selected, Text Anti-Phishing scans links in incoming text messages with the help of the Kaspersky Security Network cloud service that contains information about the reliability of files, apps, and web resources. Depending on the scan results, Text Anti-Phishing allows a text message containing a link to a safe website or prompts the user to choose an action on a text message containing a link to a malicious or spoofed website.

If the check box is cleared, Text Anti-Phishing does not scan links in incoming text messages.

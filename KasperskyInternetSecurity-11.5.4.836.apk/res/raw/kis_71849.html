Alarm

Allows / disallows remote locking of the device and turning on of a loud alarm via Web Management or using the SMS command alarm: <code> (where <code> is the secret code to Kaspersky Internet Security on your device).

Outgoing SMS commands are billed according to the price plan of the other mobile device.

If this function is allowed on the device, on receiving the Alarm command the device locks the device and turns on the alarm. If you send an SMS command, the app returns an SMS confirming that the alarm has been turned on.

Outgoing SMS messages are billed according to the price plan of your mobile service provider.

If the function is blocked on the device, the app cannot execute the Alarm command.

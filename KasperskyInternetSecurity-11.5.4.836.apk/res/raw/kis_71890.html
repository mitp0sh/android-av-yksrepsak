Changing secret code

The secret code is set when the app runs for the first time, and is used to:

  • Prevents unauthorized access to application settings
  • enable the remote launch of Anti-Theft and Privacy Protection functions from another device.

You can change the secret code.

To change the secret code:

  1. In the main window of Kaspersky Internet Security, tap Settings > Additional Settings > Change secret code.
  2. Enter the current secret code in the Enter current secret code field and tap Next.
  3. Enter a new secret code in the Create a new secret code field and tap Next.
  4. Re-enter the same code in the Re-enter the new code field and tap Enter.

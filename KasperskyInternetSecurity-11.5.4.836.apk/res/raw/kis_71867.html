Incoming call or text message

Call & Text Filter is available only on devices with an inserted SIM card.

The app prompts you to select an action to perform on receipt of an incoming SMS message or on completion of an incoming call.

The app prompts to you select an action if all of the following conditions are met: Standard mode is set for Call & Text Filter, and the contact's number is not on blocked or allowed contact lists.

You can select one of the following actions:

  • Always block.

    The app flags calls (or SMS messages) from the contact as unwanted. The app adds the number of the contact from whom the call (or SMS message) was received to the list of blocked contacts.

    If a call is received from the contact, the app blocks all subsequent incoming calls from this contact. But, if an SMS message is received from the contact, the app prompts you to select an action to perform on the incoming message.

    If an SMS message is received from the contact, the app blocks it and all subsequent SMS messages from this contact. But, if a call is received from the contact, the app prompts you to select an action to perform on the incoming call.

  • Always allow.

    The app flags calls (or SMS messages) from the contact as wanted. The app adds the number of the contact from which the call (or SMS message) was received to the list of allowed contacts.

    If a call is received from the contact, the app allows all subsequent incoming calls from this contact. But, if an SMS message is received from the contact, the app prompts you to select an action to perform on the incoming message.

    If an SMS message is received from the contact, the app allows it and all subsequent SMS messages from this contact. But, if a call is received from the contact, the app prompts you to select an action to perform on the incoming call.

  • Skip.

    The app does not take any action on the call (or SMS) from this contact. The app does not add the number of the contact from which the call (or SMS message) was received to either of the lists.

    When the next incoming call (or SMS message) is received from this contact, the app prompts you to select an action to perform on the incoming event.

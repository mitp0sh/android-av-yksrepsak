Lock and Locate settings

After performing initial configuration of Anti-Theft, you can edit settings of the Lock & Locate function.

To configure Lock & Locate:

  1. On the quick launch panel in the main window of Kaspersky Internet Security, tap Settings > Anti-Theft.
  2. Enter the secret code.

    If you forget the secret code, you can recover it.

  3. Select Allowed commands.
  4. Select or clear the Lock & Locate check box.
  5. If the Lock & Locate checkbox is set, return to the window with the Anti-Theft settings and select SIM Watch.
  6. Configure the following additional settings which are used when sending SMS commands:
    • If necessary, in the Text when locked field, enter the message to be displayed on the screen of the locked device.
    • If necessary, in the Ways to receive new phone number and device location section enter the email to which you want to receive the device location.

Blocked contacts

The list of blocked contacts contains contacts from which you do not want to receive incoming messages. For each contact, you can select the type of events to block (only SMS messages, only calls, or SMS messages and calls). For SMS messages, you can additionally enter a key phrase used to identify messages as spam and block them if a match is found.

The app blocks only those calls and text messages that satisfy all the criteria of one of the entries in the list of blocked contacts. Calls and text messages are allowed if they do not satisfy one or more criteria of the entry in the list of blocked contacts.

For example, the contact number "+123456789" has been added to the list of blocked contacts. The following filtration criteria have been selected for this contact: block only SMS with the "ad" keyword. The filter mode is set to Blocked Contacts, which causes the app to block calls and text messages from blocked contacts only and allow incoming calls and text messages from other contacts.

According to the selected filter criteria, the app blocks SMS messages from the contact that contain the word "advert", but allows calls and text messages from the contact without this key word.

The same contact with identical filter criteria cannot be included in both the blocked and allowed contacts lists. If a contact with matching filter criteria is already saved in one of the lists, the app notifies you of this fact when you try to add the contact.

To add a contact, tap Add. You can add a contact's number to the list in one of the following ways: enter the number manually, select a call from the contact in the call log, or select an SMS from the contact in SMS inbox.

You can edit or delete a contact or all contacts from the list using the context menu.

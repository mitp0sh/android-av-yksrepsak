Configure scan settings

To configure scan settings:

  1. On the quick launch panel in the main window of Kaspersky Internet Security, tap Settings > Anti-Virus > Scan.
  2. Select a value for the Type of files to scan setting.
  3. Select the action to be performed by the app on detecting a threat during scanning:
    1. Select the Disinfect check box if you want the app to automatically attempt disinfection on detecting a threat.
    2. To select the action to be taken on a threat that could not be disinfected, select the value for the If disinfection fails setting.
  4. Configure the scan frequency. Do so by selecting either Weekly or Daily for the Schedule setting.
  5. Set the day and time for the scan to start. To do so, select values for the Start day and Start time settings.

    Scans will be started according to schedule.

Type of files to scan

Selection of the type of files that the app scans for the presence of threats. App scans selected types of files when any action is performed on them (for example, when they are opened, copied, run, moved, saved, or modified). You can select one of the following file types:

  • All files: the app scans files of all types (such as music, text, system, and executable files).
  • Apps and archives only: the app scans only executable files (apps or distribution kits), such as APK files.

    When this setting is selected, the operation of Protection differs from the way it works in Recommended mode. If this setting is selected, the app can detect a threat in an app distribution kit (an APK file) when it is copied from the memory card, for example. In Recommended mode, Kaspersky Internet Security scans installed apps only and does not scan app distribution kits.

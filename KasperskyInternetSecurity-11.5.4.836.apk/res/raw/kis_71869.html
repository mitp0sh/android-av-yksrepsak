Call & Text Filter mode

This mode is enabled to block unwanted calls and text messages. The following options are available:

  • Disabled: the app does not filter calls and text messages.
  • Blocked Contacts: the app blocks calls and text messages from blocked contacts and allows calls and text messages from phone numbers that are not in the list of blocked contacts. The app deletes blocked SMS messages and stores the logs in the report.
  • Allowed Contacts: the app allows calls and text messages from allowed contacts only and blocks calls and text messages from phone numbers that are not in the list of allowed contacts. The app deletes blocked SMS messages and stores the logs in the report.
  • Standard: the app delivers calls and text messages from allowed contacts, and blocks calls and text messages from blocked contacts. If a call or SMS message is received from a number that is not in either of the lists, the app delivers it and prompts you to add the phone number to the list of blocked or allowed contacts.

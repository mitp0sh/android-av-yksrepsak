If disinfection fails

The action that the app performs on the detected file if disinfection fails or if the Disinfect check box is cleared. You can select one of the following actions:

  • Quarantine: the app moves the detected file to Quarantine.
  • Prompt for action: the app prompts you to select the action to perform on the detected file.
  • Delete: the app deletes the detected file.
  • Skip: the app does not perform any actions on the detected file and does not delete it from the device.

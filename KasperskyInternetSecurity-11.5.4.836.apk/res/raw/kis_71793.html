Cloud Security Scan

There is an additional scan of newly installed apps using the Kaspersky Security Network cloud service (a special Kaspersky Lab online service that contains information about the reliability of files, apps, and Internet resources). Scanning using the cloud service helps to protect your data from new and unknown threats.

If the Cloud Security Scan check box is selected, Kaspersky Internet Security additionally scans newly installed apps before they are run for the first time using the Kaspersky Security Network cloud service. If the check box is cleared, Kaspersky Internet Security does not carry out additional scan.

This setting is available if Protection mode is set to either Recommended or Extended.

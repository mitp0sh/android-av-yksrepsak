Update Settings

You can schedule automatic updates by choosing the frequency and the day and time of updates.

The app performs automatic scheduled updates only when the device is switched on.

Schedule

The frequency with which the app automatically runs updates. The following options are available:

  • Weekly: the app automatically starts the update once a week. For this option, you can choose the day and time to start the update.
  • Daily: the app starts the update every day. For this option, you can choose the time to start the update.
  • Disabled: the app does not automatically start a scheduled update. This allows you to run an update at any time.

    Start day

Day of starting the update. This setting is available if Weekly is selected for the Schedule setting.

Start time

Time of starting the update. This setting is available if either Daily or Weekly is selected for the Schedule setting.

Sending SMS commands

You can run Anti-Theft functions remotely or hide confidential information from any device with Kaspersky Internet Security installed. To do so, you need to send a special SMS command.

An SMS command is sent as an encrypted SMS and contains the app secret code set on the other device. Reception of the SMS command will not be noticed.

The app on the other device executes the SMS command if:

  • The device is picking up a mobile network signal;
  • Kaspersky Internet Security has been set as a Device Administrator;
  • remote launch of this function is allowed on the device.

Outgoing SMS commands are billed at your mobile service provider's current rate.

The phone number receiving the SMS command

The telephone number of the device that the SMS command is sent to.

Secret code to the app on the device receiving the SMS command

The secret code for Kaspersky Internet Security on the device receiving the SMS command.

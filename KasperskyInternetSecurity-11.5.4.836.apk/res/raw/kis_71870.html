Allow Contacts

Allow calls and text messages from all numbers in the address book (Contacts). The setting is available if the selected mode is Allowed Contacts or Standard.

If the check box is selected, the app automatically allows calls and text messages from all numbers in Contacts. If this check box is cleared, the app handles calls and text messages from numbers in Contacts according to the selected Call & Text Filter mode.

For example, the Allow Contacts check box is cleared. The selected filtering mode is Allowed Contacts. In this mode, the app allows only SMS and calls from allowed contacts, and blocks SMS and calls from all other contacts. There is not a single contact from the address book in the list of allowed contacts. In this case, the app blocks incoming events from all contacts in the address book.

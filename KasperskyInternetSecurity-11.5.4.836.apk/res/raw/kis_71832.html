Web Protection mode

The mode is used by Web Protection to scan websites before they are opened. The following modes are available:

  • Extended. Web Protection scans websites in all apps connecting to the Internet. Web Protection blocks Internet access for an app that connects to a malicious or spoofed website.
  • Standard. Web Protection only blocks websites in supported browsers and does not check them in other apps.
  • Disabled. Web Protection does not scan websites.

Web Protection scans websites before they are opened with the help of the Kaspersky Security Network cloud service that contains information about the reliability of files, apps, and web resources. Depending on the scan results, Web Protection opens reliable websites and blocks malicious websites designed to distribute malicious code, and spoofed (phishing) websites designed to steal your confidential data and gain access to your financial accounts.

Allowed contacts

The list of allowed contacts contains contacts from which you want to receive incoming messages. For each contact, you can select the type of events to allow (only SMS messages, only calls, or SMS messages and calls). For SMS messages, you can additionally enter a key phrase used to identify messages as wanted and allow them if a match is found.

The app allows only those calls and text messages that satisfy all the criteria of one of the entries in the list of allowed contacts. Calls and text messages are blocked if they do not satisfy one or more criteria of the entry in the list of allowed contacts.

For example, the contact number "+987654321" has been added to the list of allowed contacts. The following filtration criteria have been selected for this contact: allow only SMS with the "payment" keyword. The filter mode is set to Allowed Contacts, which causes the app to allow calls and text messages from allowed contacts only and block calls and text messages from other contacts.

According to the selected filter criteria, the app allows SMS messages from the contact that contain the word "payment", but blocks calls and text messages from the contact without this key word.

The same contact with identical filter criteria cannot be included in both the blocked and allowed contacts lists. If a contact with matching filter criteria is already saved in one of the lists, the app notifies you of this fact when you try to add the contact.

To add a contact, tap Add. You can add a contact's number to the list in one of the following ways: enter the number manually, select a call from the contact in the call log, or select an SMS from the contact in SMS inbox.

You can edit or delete a contact or all contacts using the context menu.

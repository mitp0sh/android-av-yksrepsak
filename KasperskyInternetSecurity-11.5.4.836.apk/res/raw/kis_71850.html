Data Wipe

Use the website or the SMS command wipe: <code> or fullreset: <code> (where <code> is your device's Kaspersky Internet Security secret code) to allow / block remote deletion of personal data or all data from the device.

Outgoing SMS commands are billed according to the price plan of the other mobile device.

Data Wipe deletes the following data:

  • Only personal data (for example, contacts, correspondence, and Googleâ„¢ account data) and memory card data.
  • All data, including memory card data. Moreover, once all the data has been deleted, the device is restored to its factory settings.

After all data has been removed and the device has been reset to factory settings, it will no longer be able to receive and perform remote commands.

If this function is allowed on the device, the app deletes personal data upon receiving a data wipe command. If you send an SMS command, the app returns an SMS confirming data deletion.

Outgoing SMS messages are billed according to the price plan of your mobile service provider.

If the function is blocked on the device, the app cannot perform the data wipe command.

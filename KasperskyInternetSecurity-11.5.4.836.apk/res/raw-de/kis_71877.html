Ferngesteuerter Start der Datenausblendung

Die Komponente Privatsphäre ist nur in der Premium-Version von Kaspersky Internet Security verfügbar. Außerdem muss auf den Geräten eine SIM-Karte eingelegt sein.

Bei Bedarf können Sie die Kontakte und dazugehörende Informationen auf dem Gerät ferngesteuert mithilfe spezieller SMS-Befehle oder über das Webportal My Kaspersky verbergen.

Die Datenausblendung über My Kaspersky steht nur für die Regionen USA und UK zur Verfügung.

Der ferngesteuerte Start der Datenausblendung ist möglich, wenn folgende Bedingungen erfüllt sind:

  • Die Premium-Version des Programms ist aktiviert.
  • Das Gerät ist zum Empfang von Mobilfunksignalen bereit.
  • Auf dem Gerät ist der ferngesteuerte Start der Funktion für die Datenausblendung erlaubt.

Um die Datenausblendung auf Ihrem Gerät ferngesteuert zu starten, führen Sie auf einem anderen Gerät eine der folgenden Aktionen aus:

  • Erstellen Sie auf dem anderen Gerät in Kaspersky Internet Security mithilfe der SMS-Befehlsfunktion einen Befehl und senden Sie ihn an Ihr Gerät (wenn das Programm auf dem anderen Gerät installiert ist).
  • Erstellen Sie auf dem anderen Gerät eine SMS mit dem speziellen Text hide:<Code> und senden Sie die SMS an Ihr Gerät. Dabei steht <Code> für den Geheimcode von Kaspersky Internet Security auf Ihrem Gerät.

Sie werden durch eine Antwort-SMS über die Befehlsausführung informiert.

Für das Senden eines SMS-Befehls an Ihr Gerät fallen Gebühren gemäß des Mobilfunktarifes an, der auf dem anderen mobilen Gerät gilt.

Um die Daten auf Ihrem Gerät ferngesteuert mithilfe des Webportals My Kaspersky auszublenden, führen Sie eine der folgenden Aktionen aus:

  1. Öffnen Sie das Webportal auf einem beliebigen Gerät.
  2. Melden Sie sich am Webportal an. Verwenden Sie dazu das Benutzerkonto für den Kaspersky Account, das Sie bei der Erstkonfiguration des Programms verwendet haben.
  3. Wählen Sie aus, auf welchem Gerät Informationen ausgeblendet werden sollen.
  4. Klicken Sie im linken Fensterbereich auf Schutzstatus.
  5. Klicken Sie unter Privatsphäre auf Kontakte verbergen.

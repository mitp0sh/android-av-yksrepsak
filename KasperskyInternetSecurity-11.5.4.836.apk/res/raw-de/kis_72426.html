Vorgehen beim Fund einer Bedrohung

Wenn Kaspersky Internet Security eine Bedrohung in einem installierten Programm erkennt, werden Sie nach dem weiteren Vorgehen für das Programm gefragt (Löschen oder Überspringen).

Wenn Kaspersky Internet Security eine Bedrohung nicht in einem Programm, sondern in einem anderen Dateityp findet (z. B. in einer Text- oder Musikdatei, in einem Archiv oder Installationspaket), so wird diese Datei standardmäßig in die Quarantäne verschoben.

Sie können festlegen, welche Aktion das Programm beim Fund einer Bedrohung ausführen soll.

Informationen über gefundene Bedrohungen finden Sie in den Abschnitten Status und Berichte.

Um Informationen über die gefundenen Bedrohungen anzusehen, führen Sie eine der folgenden Aktionen aus:

  • Klicken Sie im Hauptfenster von Kaspersky Internet Security for Android auf den Schild oder streichen Sie auf dem Bildschirm von rechts nach links (streichen Sie den Bildschirm nach links), um zum Abschnitt Status zu gelangen.
  • Klicken Sie im Hauptfenster von Kaspersky Internet Security in der Schnellstartleiste auf Einstellungen > Erweiterte Einstellungen > Berichte

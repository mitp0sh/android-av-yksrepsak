Eingehender Anruf oder SMS

Der Anruf- und SMS-Filter ist nur auf Geräten mit eingelegter SIM-Karte verfügbar.

Das Programm fordert Sie auf, eine Aktion auszuwählen, wenn eine eingehende SMS empfangen oder ein eingehender Anruf beendet wird.

Das Programm fordert Sie auf, eine Aktion auszuwählen, wenn gleichzeitig folgende Bedingungen erfüllt werden: Für den Anruf- und SMS-Filter wurde der Modus Standard festgelegt und die Nummer eines Kontakts gehört nicht zu den Listen für verbotene oder erlaubte Kontakte.

Folgende Aktionen stehen zur Auswahl:

  • Immer verbieten.

    Das Programm betrachtet Anrufe (oder SMS-Nachrichten) von diesem Kontakt als unerwünscht. Das Programm fügt die Nummer des Kontakts, von dem der Anruf (oder die SMS) stammt, zur Liste für verbotene Kontakte hinzu.

    Wenn ein Anruf von diesem Kontakt erfolgt, sperrt das Programm alle weiteren eingehenden Anrufe von diesem Kontakt. Wenn aber von diesem Kontakt künftig eine SMS empfangen wird, fordert das Programm Sie auf, eine Aktion für die eingehende SMS auszuwählen.

    Wenn von diesem Kontakt eine SMS empfangen wird, sperrt das Programm die empfangene SMS und alle folgenden SMS von diesem Kontakt. Wenn aber von diesem Kontakt künftig ein Anruf empfangen wird, fordert das Programm Sie auf, eine Aktion für den eingehenden Anruf auszuwählen.

  • Immer erlauben.

    Das Programm betrachtet Anrufe (oder SMS-Nachrichten) von diesem Kontakt als erwünscht. Das Programm fügt die Nummer des Kontakts, von dem der Anruf (oder die SMS) stammt, zur Liste für erlaubte Kontakte hinzu.

    Wenn ein Anruf von diesem Kontakt erfolgt, erlaubt das Programm alle weiteren eingehenden Anrufe von diesem Kontakt. Wenn aber von diesem Kontakt künftig eine SMS empfangen wird, fordert das Programm Sie auf, eine Aktion für die eingehende SMS auszuwählen.

    Wenn von diesem Kontakt eine SMS empfangen wird, erlaubt das Programm die empfangene SMS und alle folgenden SMS von diesem Kontakt. Wenn aber von diesem Kontakt künftig ein Anruf empfangen wird, fordert das Programm Sie auf, eine Aktion für den eingehenden Anruf auszuwählen.

  • Überspringen

    Das Programm führt für einen Anruf (oder eine SMS) von diesem Kontakt keine Aktion aus. Das Programm fügt die Nummer des Kontakts, von dem der Anruf (oder die SMS) stammt, nicht zu einer Liste hinzu.

    Wenn der nächste eingehende Anruf (oder SMS) von diesem Kontakt empfangen wird, fordert das Programm Sie erneut auf, eine Aktion für das eingehende Ereignis auszuwählen.

Geheimcode ändern

Der Geheimcode des Programms wird beim ersten Programmstart festgelegt und dient folgenden Zwecken:

  • um einen unautorisierten Zugriff auf die Programmeinstellungen zu verhindern.
  • um die Diebstahlschutz-Funktionen und das Verbergen von Kontakten ferngesteuert zu starten.

Sie können den Geheimcode ändern.

Gehen Sie folgendermaßen vor, um den Geheimcode des Programms zu ändern:

  1. Klicken Sie im Hauptfenster von Kaspersky Internet Security auf > Einstellungen > Erweiterte Einstellungen > Geheimcode ändern.
  2. Tragen Sie den aktuellen Geheimcode für das Programm im Feld Aktuellen Geheimcode eingeben ein und klicken Sie auf Weiter.
  3. Tragen Sie im Feld Legen Sie einen neuen Geheimcode fest einen neuen Geheimcode für das Programm ein und klicken Sie auf Weiter.
  4. Wiederholen Sie diesen Code im Feld Neuen Code wiederholen und klicken Sie auf OK.

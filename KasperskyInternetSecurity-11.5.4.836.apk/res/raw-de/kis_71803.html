Untersuchungsbericht

Ausführliche Informationen über eine ausgeführte Untersuchung finden Sie in den Abschnitten Status und Berichte.

Um Informationen über eine ausgeführte Untersuchung anzusehen, führen Sie eine der folgenden Aktionen aus:

  • Klicken Sie im Hauptfenster von Kaspersky Internet Security auf den Schild, um zum Abschnitt Status zu gelangen.
  • Klicken Sie im Hauptfenster von Kaspersky Internet Security in der Schnellstartleiste auf Einstellungen > Erweiterte Einstellungen > Berichte

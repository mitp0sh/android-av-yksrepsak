Cloud-Untersuchung

Zusätzliche Untersuchung neuer Programme, bevor sie zum ersten Mal gestartet werden. Dazu wird der Cloud-Dienst von Kaspersky Security Network verwendet (spezieller Online-Service von Kaspersky Lab, der Informationen über die Zuverlässigkeit von Dateien, Programmen und Internetressourcen bereitstellt). Die Untersuchung über den Cloud-Dienst dient dazu, Ihre Daten vor neuen, unbekannten Bedrohungen zu schützen.

Wenn das Kontrollkästchen Cloud-Untersuchung aktiviert ist, überprüft Kaspersky Internet Security neue Programme vor ihrem ersten Start zusätzlich mit dem Cloud-Dienst Kaspersky Security Network. Wenn dieses Kontrollkästchen deaktiviert ist, führt Kaspersky Internet Security keine zusätzliche Untersuchung durch.

Diese Einstellung ist verfügbar, wenn für Schutzmodus die Variante Empfohlen oder Erweitert ausgewählt wurde.

Auszublendende Informationen auswählen

Die Komponente Privatsphäre ist nur in der Premium-Version von Kaspersky Internet Security verfügbar. Außerdem muss auf den Geräten eine SIM-Karte eingelegt sein.

Für vertrauliche Kontakte werden vom Programm standardmäßig folgende Informationen verborgen:

  • Einträge aus dem Telefonbuch
  • gelesene eingehende SMS, gesendete SMS und SMS in den Entwürfen
  • Einträge in der Anrufliste.

Wenn eine SMS von einem privaten Kontakt eingegangen ist, zeigt das Programm die Nachricht standardmäßig im SMS-Eingangsordner an, allerdings ohne die Nummer des Kontakts anzugeben. Wenn ein vertraulicher Kontakt angerufen hat, erlaubt das Programm den eingehenden Anruf, zeigt aber die Nummer des Kontakts nicht an.

Sie können den Typ der auszublendenden Informationen ändern. Sie können beispielsweise festlegen, dass eingehende SMS und Anrufe von vertraulichen Kontakten verborgen werden. Sperren der Signale für den Eingang von SMS-Nachrichten und der eingehenden Anrufe von vertraulichen Nummern (Der Anrufer hört in diesem Fall ein "Besetzt"-Zeichen).

Um die Informationen auszuwählen, die verborgen werden sollen:

  1. Klicken Sie im Hauptfenster von Kaspersky Internet Security in der Schnellstartleiste auf Einstellungen > Privatsphäre.
  2. Geben Sie den Geheimcode für das Programm ein.

    Wenn Sie den Geheimcode vergessen haben, können Sie ihn wiederherzustellen.

  3. Setzen Sie den Schalter Privatsphäre auf deaktiviert.
  4. Klicken Sie auf  Einstellungen > Verborgene Informationen.
  5. Aktivieren Sie die Kontrollkästchen für die Informationsarten, die für private Kontakte ausgeblendet werden sollen.
  6. Kehren Sie ins Fenster Privatsphäre zurück und setzen Sie den Schalter Privatsphäre auf aktiviert.

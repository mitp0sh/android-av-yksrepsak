Über den Diebstahlschutz

Der Diebstahlschutz schützt Ihre Informationen auf dem Gerät vor unberechtigtem Zugriff und hilft Ihnen, das Gerät bei Verlust oder Diebstahl wiederzufinden.

Mithilfe des Diebstahlschutzes können Sie ferngesteuert folgende Aktionen ausführen:

  • Gerät sperren und orten
  • Alarmfunktion auf dem Gerät einschalten
  • Daten vom Gerät löschen
  • Person fotografieren, die das Gerät verwendet

Wenn die SIM-Karte auf dem Gerät ausgetauscht oder das Gerät ohne SIM-Karte eingeschaltet wird, können Sie das Gerät ferngesteuert sperren und die neue Telefonnummer ermitteln. Anschließend können Sie andere Diebstahlschutz-Funktionen auf dem vermissten Gerät starten.

Die Überwachung von Aktionen mit der SIM-Karte ist nur auf Geräten mit eingelegter SIM-Karte verfügbar.

Sie können die Diebstahlschutz-Funktionen von einem beliebigen Gerät aus mithilfe des Webportals oder mit speziellen SMS-Befehlen ferngesteuert starten. Von einem anderen mobilen Gerät aus können Sie eine Standard-SMS mit einem speziellen Text an Ihr Gerät schicken. Wenn Kaspersky Internet Security auf dem anderen Gerät installiert ist, können Sie von dort aus einen SMS-Befehl schicken. Zum Senden des SMS-Befehls muss der Geheimcode des Programms Kaspersky Internet Security bekannt sein, das auf Ihrem Gerät installiert ist.

Sie können auf Ihrem Gerät den ferngesteuerten Start der Diebstahlschutz-Funktionen erlauben oder verbieten. Wenn der Start der Diebstahlschutz-Funktion verboten ist, kann diese Funktion nicht ferngesteuert gestartet werden.

Die Einstellungen für den Diebstahlschutz sind durch einen Geheimcode geschützt (Der Code wird bei der Erstkonfiguration des Diebstahlschutzes oder beim ersten Start der Privatsphäre festgelegt).

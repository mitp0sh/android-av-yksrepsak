Schutzeinstellungen anpassen

Die Schutzeinstellungen können nur in der Premium-Version von Kaspersky Internet Security angepasst werden.

Um die Schutzeinstellungen anzupassen:

  1. Klicken Sie im Hauptfenster von Kaspersky Internet Security in der Schnellstartleiste auf Einstellungen > Anti-Virus.
  2. Wählen Sie für Schutzmodus den Wert Empfohlen oder Erweitert.
  3. Aktivieren Sie den zusätzlichen Scan mithilfe des Online-Dienstes von Kaspersky Security Network (KSN). Aktivieren Sie dazu das Kontrollkästchen Cloud-Untersuchung
  4. Aktivieren Sie die Untersuchung auf Adware, Dialer und Programme für die Fernverwaltung von Geräten. Aktivieren Sie dazu das Kontrollkästchen Adware, Autodialer u. a..

    Bei dieser Untersuchung können Programme erkannt werden, die Angreifer einsetzen können, um Schaden zu verursachen.

  5. Wenn Sie den erweiterten Schutzmodus gewählt haben, klicken Sie auf Schutz.

    Wenn Sie den empfohlenen Schutzmodus gewählt haben, ist der Abschnitt Schutz nicht verfügbar.

  6. Wählen Sie einen Dateityp für die Echtzeituntersuchung aus. Wählen Sie dazu einen Wert für Typ der zu untersuchenden Dateien.

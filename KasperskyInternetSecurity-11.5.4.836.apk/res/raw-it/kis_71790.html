Informazioni sulla scansione

È possibile avviare le seguenti scansioni:

  • Scansione completa.

    Kaspersky Internet Security esegue la scansione dell'intero file system del dispositivo.

    La scansione completa consente di proteggere i dati personali e di evitare i furti di denaro, nonché di rilevare e di eliminare le minacce nel dispositivo (sia nelle app installate che nei pacchetti di installazione).

    La maggior parte del malware mette a rischio i dati personali dell'utente presenti sul dispositivo: raccoglie i dati personali del proprietario e tutte le informazioni contenute nel dispositivo (ad esempio, le coordinate GPS e i messaggi) e li invia a un utente malintenzionato.

    Kaspersky Lab consiglia di eseguire una scansione dell'intero file system del dispositivo alla ricerca di virus almeno una volta dopo l'installazione di Kaspersky Internet Security, per assicurare la protezione dei dati personali.

  • Scansione di file e cartelle specifici.
  • Scansione rapida.

    Kaspersky Internet Security esamina solo le app installate.

    Se si utilizza la versione free, Kaspersky Lab consiglia di eseguire una scansione rapida ogni volta che si installa una nuova app.

È anche possibile configurare l'avvio automatico della scansione completa del dispositivo in base a una pianificazione creata precedentemente.

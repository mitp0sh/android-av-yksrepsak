Ripristino della password segreta

Per ripristinare la password segreta dell'app:

  1. Aprire Gestione Web in qualsiasi dispositivo.
  2. Accedere a Gestione Web con il Kaspersky Account utilizzato durante la configurazione iniziale dell'app.
  3. Selezionare il dispositivo di cui è necessario ripristinare la password segreta.
  4. Premere il pulsante Ripristino della password segreta.

    La password di ripristino viene visualizzata in Gestione Web.

  5. Avviare Kaspersky Internet Security nel dispositivo.
  6. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Impostazioni > Antifurto. 
  7. Quando viene richiesto di immettere la password segreta, toccare Menu > Ripristino della password segreta.
  8. Immettere la password di ripristino visualizzata in Gestione Web.

    La password segreta verrà visualizzata sullo schermo del dispositivo.

  9. Immettere la password segreta nel campo Immettere qui la password segreta ripristinata.

L'app aprirà la finestra Impostazioni di Antifurto.

Modalità Protezione Web

La modalità è utilizzata da Protezione Web per esaminare i siti Web prima dell'apertura. Sono disponibili le seguenti modalità:

  • Estesa. Protezione Web esamina i siti Web in tutte le app che si connettono a Internet. Protezione Web blocca l'accesso a Internet per un'app che si connette a un sito Web dannoso o contraffatto.
  • Standard. Protezione Web blocca solo i siti Web nei browser supportati e non ne esegue il controllo in altre app.
  • Disabilitato. Protezione Web non esamina i siti Web.

Protezione Web esamina i siti Web prima dell'apertura tramite il servizio cloud Kaspersky Security Network, contenente informazioni sull'attendibilità di file, app e risorse Web. A seconda dei risultati della scansione, Protezione Web apre i siti Web attendibili e blocca i siti Web dannosi che possono distribuire codice dannoso e i siti Web contraffatti (di phishing) che possono appropriarsi delle informazioni riservate dell'utente e ottenere l'accesso ai conti finanziari.

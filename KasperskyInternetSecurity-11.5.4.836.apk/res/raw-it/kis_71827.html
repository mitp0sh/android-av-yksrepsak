Informazioni su Protezione Web e Anti-Phishing messaggi di testo

Con Protezione Web e Anti-Phishing messaggi di testo, è possibile visitare i siti Web attendibili e utilizzare i dati personali in rete in modo sicuro.

Protezione Web e Anti-Phishing messaggi di testo sono disponibili solo per la versione premium di Kaspersky Internet Security.

Anti-Phishing messaggi di testo è disponibile solo per i dispositivi con una scheda SIM inserita.

Protezione Web blocca i siti Web dannosi, che tentano di distribuire codice dannoso, e i siti Web contraffatti (di phishing), che tentano di sottrarre informazioni riservate e accedere a dati finanziari.

Anti-Phishing messaggi di testo blocca i collegamenti negli SMS a siti Web dannosi o di phishing.

Protezione Web controlla i siti Web prima dell'apertura. Anti-Phishing messaggi di testo controlla inoltre i collegamenti nei messaggi SMS prima dell'apertura. Protezione Web e Anti-Phishing messaggi di testo eseguono i controlli utilizzando il servizio cloud Kaspersky Security Network (un servizio online dedicato di Kaspersky Lab che contiene informazioni sull'affidabilità di file, app e risorse Web).

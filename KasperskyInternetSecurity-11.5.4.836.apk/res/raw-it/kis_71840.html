Avvio delle funzionalità di Antifurto tramite Gestione Web

Le azioni richieste per la gestione dei dispositivi variano a seconda dell'area geografica in cui avviene l'utilizzo.

Per avviare le funzionalità di Antifurto tramite Gestione Web:

  1. Aprire Gestione Web in qualsiasi dispositivo.
  2. Accedere a Gestione Web con il Kaspersky Account utilizzato durante la configurazione iniziale dell'app.
  3. Selezionare il dispositivo per cui si desidera avviare in remoto la funzione Antifurto.
  4. Nelle sezioni presenti nella parte superiore della finestra di Gestione Web, selezionare le azioni da eseguire nel dispositivo.

Modifica della password segreta

La password segreta viene impostata durante la prima esecuzione dell'app e viene utilizzata per:

  • Impedire l'accesso non autorizzato alle impostazioni dell'applicazione
  • Abilitare l'avvio in remoto delle funzioni Antifurto e Protezione privacy da un altro dispositivo

È possibile modificare la password segreta.

Per modificare la password segreta:

  1. Nella finestra principale di Kaspersky Internet Security toccare Impostazioni > Impostazioni avanzate > Cambia password segreta.
  2. Immettere la password segreta corrente nel campo Immettere la password segreta corrente, quindi toccare Avanti.
  3. Immettere la nuova password segreta nel campo Creare una nuova password segreta, quindi toccare Avanti.
  4. Immettere nuovamente la stessa password nel campo Immettere nuovamente la nuova password, quindi toccare Invio.

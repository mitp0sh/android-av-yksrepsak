Tipi di file per la scansione

Selezione dei tipi di file esaminati dall'app alla ricerca di minacce. L'app esamina i tipi di file selezionati quando su tali file viene eseguita un'azione (ad esempio, quando vengono aperti, copiati, eseguiti, spostati, salvati o modificati). È possibile selezionare uno dei seguenti tipi di file:

  • Tutti i file: l'app esamina tutti i tipi di file (ad esempio, file di musica, di testo, di sistema ed eseguibili).
  • Solo app e archivi: l'app esamina solo i file eseguibili (app o kit di distribuzione), come i file APK.

    Se questa impostazione è selezionata, il funzionamento del componente Protezione varia rispetto alla modalità Consigliata. Se questa impostazione è selezionata, l'app è in grado di rilevare una minaccia in un kit di distribuzione dell'app (un file APK), ad esempio durante la copia dalla scheda di memoria. Nella modalità Consigliata Kaspersky Internet Security esamina solo le app installate, mentre non esamina i kit di distribuzione delle app.

Informazioni sugli aggiornamenti

Kaspersky Internet Security utilizza i database anti-virus durante la ricerca del malware. I database anti-virus dell'app contengono le descrizioni di tutti i malware attualmente noti a Kaspersky Lab e dei metodi per la loro neutralizzazione, nonché le descrizioni di altri oggetti dannosi. Si consiglia di aggiornare regolarmente i database.

Oltre ai database anti-virus, Kaspersky Internet Security consente di aggiornare la versione dell'app. Gli aggiornamenti della versione dell'applicazione rimuovono le vulnerabilità da Kaspersky Internet Security, aggiungono nuove funzioni o migliorano quelle esistenti.

Per aggiornare i database anti-virus dell'app è necessario disporre di una connessione a Internet configurata nel dispositivo.

Se si scarica Kaspersky Internet Security da Google Play, l'app aggiorna solamente i database anti-virus. La versione dell’app è stata aggiornata tramite Google Play.

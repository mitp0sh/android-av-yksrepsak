Esecuzione della configurazione iniziale di Antifurto

Le funzioni di Antifurto vengono disabilitate per impostazione predefinita dopo l'installazione dell'app: non è possibile avviarle in remoto mediante Gestione Web o i comandi SMS. Per abilitare le funzioni di Antifurto, è necessario eseguire la configurazione iniziale di Antifurto.

Al termine della configurazione iniziale, tutte le funzioni di Antifurto verranno abilitate con le impostazioni consigliate dagli specialisti di Kaspersky Lab.

La Configurazione iniziale guidata di Antifurto può essere avviata solo una volta. Successivamente è possibile configurare Antifurto nelle impostazioni dell'app.

Per eseguire la configurazione iniziale di Antifurto:

  1. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Antifurto. 

    Verrà avviata la Configurazione iniziale guidata di Antifurto.

  2. Seguire le istruzioni della procedura guidata.

Per modificare le impostazioni di Antifurto successivamente alla configurazione iniziale:

Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Impostazioni > Antifurto.

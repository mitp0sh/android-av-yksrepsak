Disinfezione dei file infetti

L'app tenta di disinfettare il file rilevato. Se la casella è selezionata, l'app tenta in primo luogo di disinfettare un file rilevato. Se l'app non è in grado di disinfettare il file rilevato, esegue l'azione specificata nella sezione Se la disinfezione fallisce.

Se la casella è deselezionata e viene rilevato un file, l'app eseguirà immediatamente l'azione specificata nella sezione Se la disinfezione fallisce.

Se la disinfezione fallisce

L'azione eseguita dall'app sul file rilevato se la disinfezione fallisce o se la casella Disinfetta è deselezionata. È possibile selezionare una delle seguenti azioni:

  • Quarantena: l'app sposta il file rilevato in Quarantena.
  • Chiedi all' utente: l'app richiede all'utente di selezionare l'azione da eseguire sul file rilevato.
  • Elimina: l'app elimina il file rilevato.
  • Ignora: l'app non esegue alcuna azione sul file rilevato e non lo elimina dal dispositivo.

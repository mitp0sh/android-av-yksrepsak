Scansione costante dei siti Web

Per abilitare la scansione costante dei siti Web:

  1. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Impostazioni > Protezione Web.
  2. Selezionare un valore per l'impostazione Modalità Protezione Web: Standard o Estesa.
  3. Toccare Cambia browser (questo pulsante viene visualizzato se il componente Protezione Web è abilitato e il browser Android standard non è impostato come browser predefinito).

    Viene avviata la procedura guidata per la selezione del browser predefinito.

  4. Seguire le istruzioni della procedura guidata.

    Al termine della procedura guidata, il browser Android standard viene impostato come browser predefinito.

Informazioni su Scansione (Anti-Virus)

Scansione (Anti-Virus) rileva e neutralizza le minacce presenti nel dispositivo.

Il componente è denominato Scansione nella versione free e Anti-Virus nella versione premium.

Scansione esegue le seguenti funzioni:

  • Scansione. Consente di avviare scansioni su richiesta dell'intero file system, solo delle app installate o di un file o di una cartella specifica.
  • Aggiornamento. Consente di scaricare database anti-virus aggiornati utilizzati per il rilevamento delle minacce. Gli aggiornamenti garantiscono il massimo livello di protezione per il dispositivo.
  • Quarantena. Consente di archiviare le minacce rilevate in un formato che non comporta rischi per il dispositivo.

Anti-Virus supporta tutte le funzionalità di Scansione e garantisce la protezione in tempo reale. Per garantire la protezione in tempo reale, Anti-Virus utilizza i database anti-virus e il servizio cloud Kaspersky Security Network.

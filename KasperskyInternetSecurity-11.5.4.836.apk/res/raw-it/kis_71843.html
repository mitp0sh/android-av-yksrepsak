Impostazioni di Cancellazione Dati

Per configurare le impostazioni di Cancellazione Dati:

  1. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Impostazioni > Antifurto.
  2. Immettere la password segreta.

    Se si dimentica la password segreta, è possibile ripristinarla.

  3. Selezionare Comandi consentiti.
  4. Selezionare o deselezionare la casella Cancellazione Dati per consentire o bloccare l'eliminazione in remoto delle informazioni personali (ad esempio, contatti, messaggi SMS, dati del calendario e dati sulla scheda di memoria) o di tutti i dati, inclusi i dati sulla scheda di memoria.

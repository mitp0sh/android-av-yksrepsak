Tipi di file per la scansione

Selezionare i file da analizzare durante la scansione del dispositivo. È possibile scegliere di esaminare i seguenti tipi di file:

  • Tutti i file: l'app esamina tutti i tipi di file (ad esempio, file di musica, di testo, di sistema ed eseguibili). Rappresenta l'impostazione predefinita.
  • Solo app e archivi: l'app esamina solo i file eseguibili (app e pacchetti di distribuzione di app), come i file APK, e non esegue la scansione di altri file. Selezionando questa opzione, la scansione del dispositivo è più veloce.

L'app esamina il contenuto degli archivi indipendentemente dal tipo di file selezionato. La scansione degli archivi in vari formati consente ad esempio di eseguire la scansione di una scheda di memoria utilizzata in diversi dispositivi.

L'app analizza i formati di archivio utilizzati in Android e altri sistemi operativi per dispositivi mobili. L'app esamina i seguenti formati di archivi: ZIP, JAR, JAD, SIS SISX, CAB e APK.

Quando l'app esamina gli archivi, li decomprime. La scansione degli archivi può pertanto ridurre in modo significativo la velocità di scansione del dispositivo.

Riferimenti aggiuntivi

È possibile utilizzare le seguenti risorse per trovare informazioni sull'applicazione:

  • Pagina dell'app nel sito Web dell'Assistenza tecnica (Knowledge Base)
  • Forum.

Se non è possibile trovare una soluzione al problema riscontrato, contattare il Servizio di Assistenza tecnica di Kaspersky Lab.

Prima di contattare il servizio di Assistenza tecnica, consultare le regole dell'assistenza tecnica.

È possibile contattare il servizio di Assistenza tecnica in uno dei modi seguenti:

  • Telefonicamente. Questo metodo consente di consultare gli specialisti dell'Assistenza tecnica in lingua russa o internazionale.
  • Inviando una query dalla propria Pagina personale Kaspersky Account sul sito Web del servizio di Assistenza tecnica. Questo metodo consente di contattare gli specialisti dell'Assistenza tecnica utilizzando un modulo di richiesta.

L'assistenza tecnica viene fornita solo agli utenti che hanno acquistato una licenza per l'utilizzo della versione premium dell'app. Gli utenti delle versioni trial non possono ottenere assistenza tecnica.

È necessario disporre di una connessione Internet per utilizzare le fonti di informazioni nel sito Web di Kaspersky Lab.

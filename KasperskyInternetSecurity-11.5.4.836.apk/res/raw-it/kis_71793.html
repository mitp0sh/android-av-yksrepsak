Scansione di sicurezza Cloud

Viene eseguita una scansione aggiuntiva delle nuove app installate utilizzando il servizio cloud Kaspersky Security Network (uno speciale servizio online di Kaspersky Lab che consente di ottenere informazioni sull'affidabilità di file, app e risorse Internet). La scansione tramite il servizio cloud consente di proteggere i dati dalle minacce nuove e sconosciute.

Se la casella Scansione di sicurezza Cloud è selezionata, Kaspersky Internet Security esegue scansioni aggiuntive delle nuove app installate prima che vengano avviate per la prima volta utilizzando il servizio cloud Kaspersky Security Network. Se la casella è deselezionata, Kaspersky Internet Security non esegue la scansione aggiuntiva.

Questa impostazione è disponibile se Modalità Protezione è impostata su Consigliata o Estesa.

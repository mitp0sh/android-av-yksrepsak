Informazioni su Antifurto

Antifurto protegge i dati dall'accesso non autorizzato e consente di localizzare il dispositivo in caso di furto o smarrimento.

Antifurto consente di eseguire le seguenti operazioni in remoto:

  • Bloccare e localizzare il dispositivo
  • Attivare un allarme nel dispositivo
  • Cancellare i dati dal dispositivo
  • Scattare una foto della persona che sta utilizzando il dispositivo

Inoltre, se la scheda SIM viene sostituita o il dispositivo viene acceso senza una scheda SIM, è possibile bloccare il dispositivo in remoto e ottenere il nuovo numero di telefono. In questo modo nel dispositivo smarrito è possibile avviare altre funzioni di Antifurto.

SIM Watch è disponibile solo per i dispositivi con una scheda SIM inserita.

È possibile attivare le funzionalità di Antifurto in remoto tramite il sito Web o utilizzando speciali comandi SMS. È possibile inviare un messaggio SMS standard con un codice speciale da un altro dispositivo mobile. Se Kaspersky Internet Security è installato nell'altro dispositivo, è possibile inviare un comando SMS dall'app. Per inviare un comando SMS, è necessario conoscere la password segreta di Kaspersky Internet Security installato nel dispositivo.

È possibile consentire o bloccare l'avvio remoto delle funzioni Antifurto sul dispositivo. Se l'avvio di Antifurto è bloccato, è impossibile avviarlo in remoto.

L'accesso alle impostazioni di Antifurto è protetto tramite una password segreta (specificata durante la configurazione iniziale di Antifurto o al momento del primo avvio di Protezione privacy).

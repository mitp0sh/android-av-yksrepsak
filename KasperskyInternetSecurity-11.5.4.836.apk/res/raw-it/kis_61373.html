Invio di comandi SMS da Kaspersky Internet Security

In caso di furto o smarrimento del dispositivo, è possibile avviare le funzioni Antifurto e Protezione privacy in remoto nel dispositivo. A tale scopo, è possibile inviare speciali comandi al dispositivo tramite Gestione Web o mediante SMS.

È possibile inviare un messaggio SMS standard con un codice speciale da un altro dispositivo mobile. Se Kaspersky Internet Security è installato nell'altro dispositivo, è possibile inviare un comando SMS dall'app. Per inviare un comando SMS, è necessario conoscere la password segreta di Kaspersky Internet Security installato nel dispositivo.

Per i comandi SMS in uscita vengono applicate le tariffe previste per l'altro dispositivo mobile.

Per avviare in remoto le funzioni Antifurto e Protezione privacy da un dispositivo in cui è installato Kaspersky Internet Security:

  1. Aprire Kaspersky Internet Security nell'altro dispositivo.
  2. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Impostazioni > Antifurto.
  3. Immettere la password segreta.

    Se si dimentica la password segreta, è possibile ripristinarla.

  4. Toccare Invia comandi SMS.
  5. Toccare Invia subito comando SMS, creare il comando e inviarlo al dispositivo. Durante la creazione del comando SMS, utilizzare la password segreta di Kaspersky Internet Security installato nel dispositivo.

Nel messaggio SMS restituito si riceverà un rapporto di esecuzione del comando.

Configurazione delle impostazioni di protezione

Le impostazioni di protezione possono essere configurate solo per la versione premium di Kaspersky Internet Security.

Per configurare le impostazioni di protezione:

  1. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Impostazioni > Anti-Virus.
  2. Selezionare un valore per l'impostazione Modalità Protezione: Consigliata o Estesa.
  3. Abilitare la scansione aggiuntiva tramite il servizio online Kaspersky Security Network (KSN). Per eseguire questa operazione, selezionare la casella Scansione di sicurezza Cloud.
  4. Abilitare la scansione per adware, auto-dialer e app di amministrazione remota. Per eseguire questa operazione, selezionare la casella Adware, auto-dialer e altro.

    Questo tipo di scansione rileva le app utilizzabili da intrusi per danneggiare gli utenti

  5. Se è stata selezionata la modalità di protezione estesa, toccare Protezione.

    Se è stata selezionata la modalità di protezione consigliata, la sezione Protezione non è disponibile.

  6. Selezionare i tipi di file da esaminare in tempo reale. A tal scopo selezionare un valore per l'impostazione Tipi di file per la scansione.
  7. Selezionare l'azione che deve essere eseguita dall'app se viene rilevata una minaccia durante la scansione in tempo reale. A tale scopo, selezionare un valore per l'impostazione Azione se viene rilevata una minaccia.

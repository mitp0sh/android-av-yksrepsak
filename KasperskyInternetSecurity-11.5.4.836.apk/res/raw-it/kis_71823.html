Informazioni sulla Quarantena

L'app sposta i file rilevati nel corso delle scansioni del dispositivo o durante l'esecuzione del componente Protezione in Quarantena (un archivio isolato dedicato). L'app archivia i file in Quarantena in un formato compresso, in modo che non possano danneggiare il dispositivo.

È possibile visualizzare i file spostati in Quarantena. È anche possibile eliminare o ripristinare i file utilizzando il menu di scelta rapida.

Quando un file viene eliminato dalla Quarantena, non può essere ripristinato. Quando un file viene ripristinato dalla Quarantena, l'app lo sposta nella cartella originale da cui era stato rimosso.

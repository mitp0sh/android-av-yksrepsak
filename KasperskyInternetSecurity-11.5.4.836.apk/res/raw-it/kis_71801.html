Avvio di una scansione

Per avviare una scansione:

  1. Nel riquadro di avvio veloce della finestra principale di Kaspersky Internet Security toccare Scansione.
  2. Selezionare uno dei tipi di scansione seguenti:
    • Scansione completa: consente di eseguire la scansione dell'intero file system del dispositivo.
    • Scansione rapida: consente di esaminare solo le app installate.
    • Scansione cartella: consente di esaminare un file o una cartella specifica.

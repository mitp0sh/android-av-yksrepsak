Параметры обновления

Вы можете настроить автоматический запуск обновления по расписанию: выбрать периодичность запуска, день и время запуска.

Программа выполняет автоматическое обновление по расписанию только, когда устройство включено.

Расписание

Периодичность, в соответствии с которой программа будет автоматически запускать обновление. Предусмотрены следующие варианты:

  • Раз в неделю – программа автоматически запускает обновление раз в неделю. Для этого варианта вы можете выбрать день и время запуска обновления.
  • Раз в день – программа запускает обновление каждый день. Для этого варианта вы можете выбрать время запуска обновления.
  • Не установлено – программа автоматически не запускает обновление по расписанию. При этом вы можете запускать обновление в любое удобное для вас время.

    День запуска

День запуска обновления. Параметр доступен, если для параметра Расписание выбрано значение Раз в неделю.

Время запуска

Время запуска обновления. Параметр доступен, если для параметра Расписание выбрано значение Раз в день или Раз в неделю.

Настройка параметров защиты

Настройка параметров защиты доступна только в полной версии Kaspersky Internet Security.

Чтобы настроить параметры защиты:

  1. В главном окне Kaspersky Internet Security в панели быстрого запуска нажмите Настройка > Антивирус.
  2. Выберите для параметра Режим защиты значение рекомендуемый или расширенный.
  3. Включите дополнительную проверку с помощью онлайн-сервиса Kaspersky Security Network (KSN). Для этого установите флажок Проверка в облаке.
  4. Включите проверку на наличие рекламных программ, а также программ автодозвона или программ для удаленного управления устройством. Для этого установите флажок Реклама, автодозвон и другое.

    Такая проверка позволяет обнаружить программы, которыми могут воспользоваться злоумышленники для причинения вреда пользователям.

  5. Если вы выбрали расширенный режим защиты, нажмите Защита.

    Если вы выбрали рекомендуемый режим защиты, блок Защита будет недоступен.

  6. Выберите тип файлов для проверки в реальном времени. Для этого выберите значение для параметра Тип проверяемых файлов.
  7. Выберите действие программы при обнаружении угрозы во время проверки в реальном времени. Для этого выберите значение для параметра Действие при обнаружении угрозы.

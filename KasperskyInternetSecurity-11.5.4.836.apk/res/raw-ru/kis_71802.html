Настройка параметров проверки

Чтобы настроить параметры проверки:

  1. В главном окне Kaspersky Internet Security в панели быстрого запуска нажмите Настройка > Антивирус > Проверка.
  2. Выберите значение для параметра Тип проверяемых файлов.
  3. Выберите действие программы при обнаружении угрозы во время проверки:
    1. Установите флажок Лечить, чтобы при обнаружении угрозы программа автоматически пыталась ее вылечить.
    2. Чтобы выбрать действие программы с угрозой, которую не удалось вылечить, выберите значение для параметра Если лечение невозможно.
  4. Настройте периодичность запуска проверки. Для этого выберите для параметра Расписание значение Раз в неделю или Раз в день.
  5. Настройте день и время запуска проверки. Для этого выберите значения для параметров День запуска и Время запуска.

    Проверка будет запускаться согласно расписанию.

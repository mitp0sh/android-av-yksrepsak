Запуск функций Анти-Вора через веб-портал

В зависимости от региона действия на веб-портале для управления устройством могут различаться.

Чтобы запустить функции Анти-Вора через веб-портал:

  1. Откройте веб-портал на любом устройстве.
  2. Войдите на веб-портал с помощью вашей учетной записи Kaspersky Account, которую вы использовали при первоначальной настройке программы.
  3. Выберите устройство, для которого вам нужно дистанционно запустить функции Анти-Вора.
  4. В блоках в верхней части окна веб-портала выберите действия, которые нужно выполнить на устройстве.

Продление срока действия лицензии

Kaspersky Internet Security проверяет наличие лицензии на использование полной версии Kaspersky Internet Security при каждом запуске программы.

Если вы используете полную версию программы, вы можете узнать количество дней, оставшихся до окончания срока действия лицензии, посмотрев информацию о лицензии.

Когда до окончания срока действия лицензии осталось менее 14 дней, вы можете его продлить. Для этого вы можете воспользоваться одним из следующих способов:

  • Приобрести новую лицензию в интернет-магазине.
  • Продлить срок действия лицензии на веб-портале My Kaspersky.

    Веб-портал My Kaspersky доступен только для регионов США и Великобритании.

  • Использовать код активации, заранее полученный при приобретении лицензии.

По истечении срока действия лицензии программа автоматически переходит на бесплатную версию.

Чтобы продлить срок действия лицензии с помощью кода активации:

  1. В главном окне Kaspersky Internet Security в панели быстрого запуска нажмите Настройка > Дополнительные параметры > Лицензирование > Продлить срок действия.

    Программа может предложить использовать лицензию, обнаруженную для вашей учетной записи на веб-портале My Kaspersky. В этом случае откажитесь от активации по обнаруженной лицензии, нажав Другие способы активации.

    Веб-портал My Kaspersky доступен только для регионов США и Великобритании.

    Если для вашей учетной записи программа не обнаружит лицензии, по которой можно активировать программу, сразу перейдите к следующему шагу.

  2. Выберите Ввод кода активации.
  3. Введите код активации, заранее полученный при приобретении лицензии.

Изменение секретного кода

Секретный код программы устанавливается при первом запуске программы и используется в следующих целях:

  • чтобы предотвратить несанкционированный доступ к параметрам программы;
  • чтобы дистанционно с другого устройства запустить функции Анти-Вора и скрытие контактов.

Вы можете изменить секретный код.

Чтобы изменить секретный код программы, выполните следующие действия:

  1. В главном окне Kaspersky Internet Security нажмите Настройка > Дополнительные параметры > Изменение секретного кода.
  2. Введите текущий секретный код программы в поле Введите текущий секретный код и нажмите Далее.
  3. Введите новый секретный код программы в поле Придумайте новый секретный код и нажмите Далее.
  4. Введите этот код еще раз в поле Повторно введите новый код и нажмите Вход.

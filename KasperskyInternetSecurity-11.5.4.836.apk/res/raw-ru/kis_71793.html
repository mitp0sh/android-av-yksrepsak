Проверка в облаке

Дополнительная проверка новых приложений до их первого запуска при помощи облачного сервиса Kaspersky Security Network (специального онлайн-сервиса "Лаборатории Касперского", который содержит информацию о надежности файлов, приложений и интернет-ресурсов). Проверка при помощи облачного сервиса помогает защитить ваши данные от новых неизвестных угроз.

Если флажок Проверка в облаке установлен, Kaspersky Internet Security дополнительно проверяет новые приложения до их первого запуска при помощи облачного сервиса Kaspersky Security Network. Если флажок снят, Kaspersky Internet Security не выполняет дополнительную проверку.

Параметр доступен, если для параметра Режим защиты выбран вариант Рекомендуемый или Расширенный.

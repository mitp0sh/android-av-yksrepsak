Тип проверяемых файлов

Выбор типа файлов, которые программа проверяет на наличие угроз. Программа проверяет файлы выбранного типа при любом действии над ними (например, при их открытии, копировании, запуске, перемещении, сохранении или изменении). Вы можете выбрать один из следующих типов файлов:

  • Все файлы – программа проверяет файлы всех типов (например, музыкальные, текстовые, системные, исполняемые).
  • Только программы и архивы – программа проверяет только исполняемые файлы (то есть программы или дистрибутивы программ), например, файлы формата APK.

    Работа защиты при выборе этого параметра отличается от работы в рекомендуемом режиме. Если выбран этот параметр, то защита может обнаружить угрозу в дистрибутиве программы (файле формата APK), например, при его копировании с карты памяти. В рекомендуемом режиме Kaspersky Internet Security проверяет только установленные программы и не проверяет дистрибутивы программ.

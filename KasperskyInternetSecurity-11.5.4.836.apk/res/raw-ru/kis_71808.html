Тип проверяемых файлов

Выбор файлов, которые программа анализирует во время проверки файловой системы устройства. Вы можете выбрать для проверки файлы следующих типов:

  • Все файлы – программа проверяет файлы всех типов (например, музыкальные, текстовые, системные, исполняемые). Параметр выбран по умолчанию.
  • Только программы и архивы – программа проверяет только исполняемые файлы (то есть программы или дистрибутивы программ), например, файлы формата APK, и не проверяет другие файлы. При выборе этого варианта скорость проверки устройства увеличивается.

Независимо от типа выбранных файлов программа проверяет содержимое архивов. Проверка архивов разных форматов позволяет, например, проверять карту памяти, которая используется в нескольких устройствах.

Программа анализирует архивы тех форматов, которые применяются в Android и других мобильных операционных системах. Программа проверяет архивы следующих форматов: ZIP, JAR, JAD, SIS, SISX, CAB и APK.

Когда программа проверяет архивы, она их распаковывает. Поэтому проверка архивов может значительно снизить скорость выполнения проверки устройства.

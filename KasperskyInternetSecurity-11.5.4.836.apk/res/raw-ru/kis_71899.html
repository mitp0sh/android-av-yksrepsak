Активация полной версии программы

Чтобы использовать все функции программы, вам нужно активировать полную версию Kaspersky Internet Security.

Активация – это перевод программы в полнофункциональный режим.

Для активации вы можете использовать один из следующих способов:

  • Ввести код активации, который вы получили при приобретении лицензии.
  • В программе при активации использовать лицензию, обнаруженную для вашей учетной записи на веб-портале My Kaspersky, или открыть веб-портал My Kaspersky и с помощью него активировать программу.

    Веб-портал My Kaspersky доступен только для регионов США и Великобритании.

  • Приобрести лицензию в интернет-магазине.

Вы можете активировать полную версию программы при первом запуске или в любое время позже.

Для активации программы нужно активное интернет-соединение.

Чтобы активировать полную версию программы с помощью кода активации:

  1. В главном окне Kaspersky Internet Security в панели быстрого запуска нажмите Настройка > Дополнительные параметры > Лицензирование > Активировать полную версию.

    Программа может предложить использовать лицензию, обнаруженную для вашей учетной записи на веб-портале My Kaspersky. В этом случае откажитесь от активации по обнаруженной лицензии, нажав Другие способы активации.

    Веб-портал My Kaspersky доступен только для регионов США и Великобритании.

  2. Выберите Ввод кода активации.
  3. Введите код активации в открывшемся окне и нажмите Активировать.

Программа отправит запрос на активацию на сервер активации "Лаборатории Касперского". При успешном выполнении запроса на активацию программа уведомит вас об этом и покажет информацию о лицензии.

Чтобы активировать полную версию программы по лицензии, обнаруженной для вашей учетной записи на веб-портале My Kaspersky, выполните следующие действия:

Этот способ активации доступен только для регионов США и Великобритании.

  1. В главном окне Kaspersky Internet Security в панели быстрого запуска нажмите Настройка > Дополнительные параметры > Лицензирование > Активировать полную версию.

    Если для вашей учетной записи на веб-портале My Kaspersky обнаружена лицензия, по которой можно активировать программу, Kaspersky Internet Security предложит вам использовать ее для активации.

    Если для вашей учетной записи программа не обнаружит лицензии, по которой можно активировать программу, Kaspersky Internet Security предложит вам другие способы активации.

  2. Выберите нужную лицензию и нажмите Продолжить.

Программа отправит запрос на активацию на сервер активации "Лаборатории Касперского". При успешном выполнении запроса на активацию программа уведомит вас об этом и покажет информацию о лицензии.

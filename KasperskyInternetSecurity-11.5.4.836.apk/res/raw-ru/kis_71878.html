Автоматическое скрытие данных

Компонент Личные контакты доступен только в полной версии Kaspersky Internet Security и на устройствах с установленной SIM-картой.

Для действий с Личными контактами (просмотра SMS, вызовов от конфиденциальных контактов, настройки параметров) необходимо выключать скрытие информации. После выполнения действий с Личными контактами скрытие информации нужно включать повторно. Вы можете включить автоматическое скрытие информации после действий с Личными контактами через указанное время (после перехода устройства в режим энергосбережения).

Чтобы включить автоматическое скрытие информации о конфиденциальных контактах:

  1. В главном окне Kaspersky Internet Security в панели быстрого запуска нажмите Настройка > Личные контакты. 
  2. Введите секретный код программы.

    Если вы забыли секретный код, вы можете его восстановить.

  3. Переведите переключатель Личные контакты в состояние выключено.
  4. Нажмите Настройка.
  5. Выберите время, через которое нужно автоматически включать скрытие информации. Для этого выберите значение для параметра Автоматическое скрытие.

Когда скрытие информации выключено, программа будет автоматически включать его через указанное вами время после перехода устройства в режим энергосбережения.

Cloud Security Scan

Existe un análisis adicional de las aplicaciones recién instaladas que utiliza el servicio en la nube de Kaspersky Security Network (un servicio en línea especial de Kaspersky Lab que contiene información sobre la fiabilidad de los archivos, las aplicaciones y los recursos de Internet). El análisis mediante el servicio en la nube lo ayuda a proteger sus datos frente a amenazas nuevas y desconocidas.

Si se activa la casilla de verificación Cloud Security Scan, Kaspersky Internet Security realizará comprobaciones adicionales mediante el servicio en la nube de Kaspersky Security Network en las aplicaciones recién instaladas antes de que se inicien por primera vez. Si se desactiva la casilla, Kaspersky Internet Security no realizará análisis adicional.

Esta configuración está disponible si el modo de protección está establecido en Recomendado o Ampliado.

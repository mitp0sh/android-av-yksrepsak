Ejecución planificada de un análisis

Puede configurar una planificación para que se inicie automáticamente un análisis completo del dispositivo: elija la frecuencia con la que se iniciará el análisis, así como el día y la hora (opcional).

Para que la aplicación ejecute el análisis automáticamente, el dispositivo debe estar encendido en el momento en que esté programado el inicio del análisis.

La aplicación iniciará el análisis planificado automático en segundo plano. Cuando se detecta un objeto malintencionado, la aplicación lleva a cabo la acción seleccionada en la configuración del análisis.

Planificación

Frecuencia con la que se ejecuta un análisis completo del sistema de archivos. Están disponibles las siguientes opciones:

  • Semanalmente: La aplicación inicia automáticamente el análisis una vez a la semana. En esta opción, puede elegir el día y la hora de inicio del análisis.
  • A diario: La aplicación inicia automáticamente el análisis cada día a la hora especificada. En esta opción, puede elegir la hora de inicio del análisis.
  • Desactivado: La aplicación no realizará un análisis planificado automático. Esta opción le permite iniciar un análisis en el momento.

 

Día de inicio

Día del análisis. Este parámetro estará disponible si se selecciona Semanalmente en el parámetro Planificación.

Hora de inicio

Hora del análisis. Este parámetro estará disponible si se selecciona A diario o Semanalmente en el parámetro Planificación.

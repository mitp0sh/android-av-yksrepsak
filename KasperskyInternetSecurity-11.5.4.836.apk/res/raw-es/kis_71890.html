Modificación del código secreto

El código secreto se establece cuando se inicia la aplicación por primera vez y se utiliza para:

  • Evita el acceso no autorizado a la configuración de la aplicación.
  • Activar el inicio remoto de las funciones Antirrobo y Protección de privacidad desde otro dispositivo.

Es posible modificar el código secreto.

Para modificar el código secreto:

  1. En la ventana principal de Kaspersky Internet Security, toque Configuración > Configuración adicional > Cambiar código secreto.
  2. Introduzca el código secreto actual en el campo Introduzca el código secreto actual y toque Siguiente.
  3. Introduzca el nuevo código secreto en el campo Cree un nuevo código secreto y toque Siguiente.
  4. Vuelva a escribir el mismo código en el campo Vuelva a introducir el nuevo código y toque Entrar.

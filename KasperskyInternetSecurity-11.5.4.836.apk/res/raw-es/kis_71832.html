Modo Protección web

Protección web usa este modo para analizar sitios web antes de abrirlos. Están disponibles los siguientes modos:

  • Ampliado. Protección web analiza sitios web en todas las aplicaciones que se conectan a Internet. Protección web bloquea el acceso a Internet para una aplicación que se conecte a un sitio web malintencionado o falsificado.
  • Estándar. Protección web solo bloquea sitios web en navegadores compatibles y no en otras aplicaciones.
  • Desactivado. Protección web no analiza sitios web.

Protección web analizará sitios web antes de abrirlos gracias al servicio en la nube de Kaspersky Security Network, que contiene información sobre el grado de fiabilidad de los archivos, las aplicaciones y los recursos web. En función de los resultados del análisis, Protección web abrirá sitios web fiables y bloqueará sitios web malintencionados, diseñados para distribuir código malintencionado, así como sitios web fraudulentos (de suplantación de identidad) diseñados para robar sus datos confidenciales y para acceder a sus cuentas bancarias.

Envío de comandos SMS desde Kaspersky Internet Security

En caso de pérdida o robo del dispositivo, puede iniciar las funciones Antirrobo o Protección de privacidad de forma remota en el dispositivo. Para ello, envíe comandos especiales a su dispositivo a través de Gestión web o por SMS.

Puede enviar un mensaje SMS estándar con un código especial desde otro dispositivo móvil. También puede enviar un comando SMS desde la aplicación si Kaspersky Internet Security está instalado en el otro dispositivo. Para crear un comando SMS, tendrá que saber el código secreto de Kaspersky Internet Security instalado en su dispositivo.

Los comandos SMS enviados se facturan de acuerdo con el plan de precios del otro dispositivo.

Para iniciar las funciones Antirrobo y Protección de privacidad de forma remota desde un dispositivo con Kaspersky Internet Security instalado:

  1. Abra Kaspersky Internet Security en el otro dispositivo.
  2. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Antirrobo.
  3. Introduzca el código secreto.

    Si ha olvidado su código secreto, puede recuperarlo.

  4. Toque Enviar comandos SMS.
  5. Toque Enviar comando SMS ahora, cree el comando y envíelo al dispositivo. Al crear el comando SMS, use el código secreto de Kaspersky Internet Security instalado en su dispositivo.

Recibirá un informe de ejecución del comando en el mensaje SMS de respuesta.

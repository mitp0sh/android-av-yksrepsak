Información de suministro de datos

Para mejorar la protección operativa, los siguientes componentes de Kaspersky Internet Security utilizan el servicio en la nube Kaspersky Security Network:

  • Analizar. La aplicación también analiza las aplicaciones instaladas antes de su primera ejecución. Este análisis permite detectar nuevas amenazas que todavía no se han descrito en las bases de datos antivirus.
  • Protección web. La aplicación también analiza los sitios web antes de abrirlos.
  • Antifraude de mensajes de texto. La aplicación bloquea vínculos en mensajes de texto que dirijan a sitios web malintencionados o fraudulentos. 

Para obtener más información acerca de los datos que se transfieren a Kaspersky Lab al usar el servicio en la nube cuando Kaspersky Internet Security está instalado en el dispositivo del usuario, consulte el Contrato de licencia. Al aceptar los términos y condiciones del Contrato de licencia, acepta transferir automáticamente a Kaspersky Lab la siguiente información:

  • conexiones inalámbricas;
  • hardware y software utilizado en el dispositivo;
  • sumas de los archivos procesados (MD5 y SHA-256);
  • tipos de amenazas detectadas, resultados de su procesamiento y bases de datos antivirus instaladas;
  • URL de sitios web para evaluar su reputación, identificador único del protocolo de conexión y su número de puerto;
  • datos de activación de Kaspersky Internet Security y la versión en uso;
  • identificador único del dispositivo;
  • resultados del análisis heurístico;
  • identificador único del usuario;
  • identificador de la conexión al portal web y su estado de conexión;
  • aplicaciones instaladas.

También es posible que se envíen a Kaspersky Lab para su análisis los archivos (y componentes) susceptibles de ser utilizados por parte de intrusos para dañar el dispositivo o sus datos personales.

Kaspersky Lab protege toda la información recopilada por este medio según lo establecido por ley. Kaspersky Lab usa la información obtenida únicamente para estadísticas generales. Las estadísticas generales se generan automáticamente a partir de la información original recibida y no contienen datos personales ni confidenciales. La información original recibida se almacena cifrada y se elimina a medida que se acumula (dos veces al año). Los datos de estadísticas generales se almacenan de forma indefinida.

Acción al detectar una amenaza

Al detectar una amenaza en una aplicación instalada, Kaspersky Internet Security le solicita que elija la acción que se llevará a cabo para esta aplicación (eliminar u omitir).

Si Kaspersky Internet Security detecta un amenaza en un archivo distinto a una aplicación (como un archivo de texto o audio, un archivo comprimido o un kit de distribución), este archivo se mueve a Cuarentena de manera predeterminada.

Puede configurar la acción que llevará a cabo la aplicación de manera predeterminada al detectar una amenaza.

Puede ver información sobre las amenazas detectadas en las secciones Estado e Informes.

Para ver información acerca de las amenazas detectadas, realice una de las operaciones siguientes:

  • En la ventana principal de Kaspersky Internet Security, toque el escudo o deslícese por la pantalla de derecha a izquierda para abrir la sección Estado.
  • En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Configuración adicional > Informes.

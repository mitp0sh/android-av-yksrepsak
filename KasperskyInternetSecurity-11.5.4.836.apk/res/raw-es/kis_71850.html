Borrado de datos

Utilice el sitio web o el comando SMS wipe: <código> o fullreset: <código> (donde <código> es el código secreto para Kaspersky Internet Security en su dispositivo) para permitir o bloquear la eliminación de forma remota de los datos personales o de todos los datos del dispositivo.

Los comandos SMS enviados se facturan de acuerdo con el plan de precios del otro dispositivo.

Borrado de datos elimina los datos siguientes:

  • Datos personales (como contactos, correspondencia y datos de la cuenta de Googleв„ў) y datos de la tarjeta de memoria.
  • Todos los datos, incluidos los de la tarjeta de memoria. Además, una vez eliminados todos los datos, el dispositivo se restaura con los ajustes de fábrica.

Después de que se hayan eliminado todos los datos y el dispositivo se haya restablecido a la configuración de fábrica, ya no podrá recibir ni ejecutar comandos remotos.

La aplicación elimina los datos personales al recibir un comando de borrado de datos si se ha autorizado esta función en el dispositivo. La aplicación devuelve un SMS de confirmación del borrado si envía un comando SMS.

Los mensajes SMS enviados se facturan de acuerdo con el plan de precios de su operador de servicios móviles.

Si la función está bloqueada en el dispositivo, la aplicación no podrá llevar a cabo el borrado de datos.

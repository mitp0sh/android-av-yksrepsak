Funciones principales de la aplicación

Kaspersky Internet Security ofrece las siguientes funciones principales:

Protección frente a virus y otros tipos de software malintencionado

El componente Scanner (Antivirus) protege el dispositivo frente a virus y otros tipos de software malintencionado.

El componente se llama Scanner en la versión gratuita y Antivirus en la versión Premium.

Scanner le permite analizar todo el dispositivo, las aplicaciones instaladas o una carpeta específica en busca de amenazas, configurar análisis programados del dispositivo y actualizar las bases de datos antivirus para disfrutar de una protección actualizada de sus datos.

Antivirus combina todas las funcionalidades de Scanner con las siguientes funciones adicionales:

  • Protege su dispositivo en tiempo real.
  • Analiza las aplicaciones recién instaladas antes de que las ejecute por primera vez, por medio de bases de datos antivirus y el servicio en la nube en línea de Kaspersky Security Network.
  • Admite actualizaciones automáticas de las bases de datos antivirus.

Permite ocultar contactos confidenciales junto con su historial de llamadas e intercambio de SMS.

El componente Protección de privacidad oculta los contactos confidenciales y la información relacionada con ellos.

Protección de privacidad está disponible únicamente con la versión Premium de Kaspersky Internet Security en dispositivos con una tarjeta SIM insertada.

Protección de privacidad le permite ocultar temporalmente los contactos confidenciales y todo lo relacionado con el historial de llamadas y la correspondencia de SMS. Puede ocultar los contactos y la información relacionada mediante la configuración de la aplicación o remotamente a través de comandos SMS especiales.

Protección de datos en caso de pérdida o robo del aparato

Antirrobo protege sus datos contra el acceso no autorizado y ayuda a localizar el dispositivo en caso de pérdida o robo.

Antirrobo le permite activar la alarma del dispositivo, bloquear el dispositivo, determinar su ubicación, borrar datos del dispositivo y hacer una foto de identificación a la persona que está usando el dispositivo en esos momentos de forma remota. Antirrobo le permite iniciar de forma remota las funciones del dispositivo mediante comandos especiales de SMS o a través de Gestión web. En dispositivos con ranura para tarjeta SIM, si la tarjeta se sustituye por otra o el dispositivo se enciende sin una tarjeta SIM, Antirrobo puede enviarle el nuevo número de teléfono del dispositivo en un mensaje SMS o un correo electrónico, o bloquear el dispositivo.

Bloqueo de llamadas y de mensajes de texto no solicitados

Filtro de llamadas y SMS se utiliza para bloquear llamadas y mensajes de texto no solicitados.

El componente Filtro de llamadas y SMS solo está disponible en dispositivos con una tarjeta SIM insertada.

Protección contra amenazas en línea

Los componentes Protección web y Antifraudes de mensajes de texto ofrecen protección frente a amenazas en línea.

Los componentes Protección web y Antifraudes de mensajes de texto están disponibles únicamente en la versión Premium de Kaspersky Internet Security.

 

El componente Antifraudes de mensajes de texto solo está disponible en dispositivos con una tarjeta SIM insertada.

Protección web bloquea sitios web malintencionados que distribuyen código malicioso y sitios web fraudulentos que roban datos confidenciales y obtienen acceso a sus cuentas financieras.

Antifraudes de mensajes de texto bloquea vínculos de SMS que dirijan a sitios web malintencionados o fraudulentos.

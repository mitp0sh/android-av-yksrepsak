Recuperación del código secreto

Para recuperar el código secreto de la aplicación de modo seguro:

  1. Abra Gestión web desde cualquier dispositivo.
  2. Inicie sesión en Gestión web con la cuenta de Kaspersky que indicó en la configuración inicial de la aplicación.
  3. Seleccione el dispositivo cuyo código secreto debe recuperarse.
  4. Presione el botón Recuperación del código secreto.

    El código de recuperación aparece en Gestión web.

  5. Inicie Kaspersky Internet Security en el dispositivo.
  6. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Antirrobo. 
  7. Cuando se le solicite introducir el código secreto, toque Menú > Recuperación del código secreto.
  8. Introduzca el código de recuperación mostrado en Gestión web.

    El código secreto se muestra en la pantalla del dispositivo.

  9. Introduzca el código secreto en el campo Introduzca aquí el código secreto restaurado.

La aplicación abrirá la ventana Configuración de Antirrobo.

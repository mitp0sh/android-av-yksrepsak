Información de actualizaciones

Kaspersky Internet Security utiliza bases de datos antivirus cuando realiza búsquedas de software malintencionado. Las bases de datos antivirus de la aplicación contienen descripciones del software malintencionado que conozca actualmente Kaspersky Lab, las maneras de neutralizarlo y descripciones de otros objetos malintencionados. Recomendamos actualizar periódicamente las bases de datos de la aplicación.

Al igual que con las bases de datos antivirus, Kaspersky Internet Security le permite actualizar la versión de la aplicación. Las actualizaciones de la versión de la aplicación eliminan vulnerabilidades de Kaspersky Internet Security, agregan nuevas funciones o mejoran las actuales.

Para actualizar las bases de datos antivirus de la aplicación, debe disponer de una conexión a Internet ya configurada en su dispositivo.

Si descarga Kaspersky Internet Security desde Google Play, la aplicación solo actualizará las bases de datos antivirus. La versión de la aplicación se actualiza a través de Google Play.

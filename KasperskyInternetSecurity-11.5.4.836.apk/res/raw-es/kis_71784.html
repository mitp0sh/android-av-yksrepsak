Configuración de la protección

La configuración de protección solo está disponible con la versión Premium de Kaspersky Internet Security.

Para configurar los parámetros de protección:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Antivirus.
  2. Seleccione un valor para el Modo de protección Recomendado o Ampliado.
  3. Activar análisis adicionales con la ayuda del servicio en línea de Kaspersky Security Network (KSN). Para ello, seleccione la casilla de verificación Cloud Security Scan.
  4. Active el análisis de software publicitario, marcadores automáticos y aplicaciones de gestión remotas. Para ello, seleccione la casilla de verificación Software publicitario, marcadores automáticos y otros.

    Este tipo de análisis detecta las aplicaciones de las que se pueden aprovechar los intrusos para dañar a los usuarios.

  5. Si ha seleccionado el modo de protección Ampliado, toque Protección.

    Si ha seleccionado el modo de protección Recomendado, la sección Protección no se encuentra disponible.

  6. Seleccione los tipos de archivos que se van a analizar en tiempo real. Para ello, seleccione un valor para el ajuste Tipo de archivos para analizar.
  7. Seleccione la acción que la aplicación debe realizar al detectar una amenaza durante el análisis en tiempo real. Para ello, seleccione un valor para el parámetro de configuración Acción al detectar una amenaza.

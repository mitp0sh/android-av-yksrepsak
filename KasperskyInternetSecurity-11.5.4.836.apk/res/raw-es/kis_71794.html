Software publicitario, marcadores automáticos y otros

Detección de software publicitario y aplicaciones que pueden utilizar los delincuentes para dañar su dispositivo o sus datos personales.

Las siguientes aplicaciones pertenecen a este tipo de software:

  • Marcadores automáticos
  • Aplicaciones que admiten mensajes SMS salientes de pago
  • Aplicaciones para la gestión remota del dispositivo
  • Aplicaciones para supervisar la actividad del usuario en el dispositivo (por ejemplo, aplicaciones que solicitan el acceso a sus datos personales, envían mensajes de texto o determinan su ubicación)

Si la casilla de verificación está seleccionada, Kaspersky Internet Security detecta aplicaciones que pertenezcan a estas categorías y solicita al usuario que realice acciones al respecto. Si se desactiva esta casilla, Kaspersky Internet Security omite las aplicaciones pertenecientes a estas categorías.

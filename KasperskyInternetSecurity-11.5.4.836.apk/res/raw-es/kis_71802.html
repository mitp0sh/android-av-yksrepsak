Configuración de los parámetros del análisis

Para configurar los parámetros del análisis:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Antivirus > Analizar.
  2. Seleccione un valor para el ajuste Tipo de archivos para analizar.
  3. Seleccione la acción que efectuará la aplicación al detectar una amenaza durante el análisis:
    1. Seleccione la casilla de verificación Desinfectar si desea que la aplicación trate de desinfectar la amenaza automáticamente al detectarla.
    2. Para seleccionar la acción que se llevará a cabo sobre una amenaza que no se pudo desinfectar, seleccione el valor del parámetro Si falla la desinfección.
  4. Configure la frecuencia del análisis. Para ello, seleccione las opciones Semanalmente o A diario para el parámetro de configuración Planificación.
  5. Defina el día y la hora de inicio del análisis. Para ello, seleccione los valores para la configuración de Día de inicio y Hora de inicio.

    Los análisis comenzarán según la planificación.

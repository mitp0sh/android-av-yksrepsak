Información de la función Antirrobo

Antirrobo protege sus datos frente al acceso no autorizado y ayuda a localizar el dispositivo en caso de pérdida o sustracción.

Antirrobo le permite realizar las siguientes acciones de forma remota:

  • Bloquear el dispositivo y determinar su ubicación.
  • Activar una alarma ruidosa en el dispositivo.
  • Borrar datos del dispositivo.
  • Tomar una foto de identificación de la persona que esté usando el dispositivo actualmente.

Asimismo, si se sustituye la tarjeta SIM o se enciende el dispositivo sin esta, podrá bloquear remotamente el dispositivo y obtener el número de teléfono nuevo. Esto le permite iniciar otras funciones de Antirrobo en el dispositivo perdido.

Detector de SIM está disponible solo para dispositivos que dispongan de tarjeta SIM.

Puede iniciar Antirrobo de forma remota desde cualquier dispositivo por medio del sitio web o mediante comandos SMS especiales. Puede enviar un mensaje SMS estándar con un código especial desde otro dispositivo móvil. También puede enviar un comando SMS desde la aplicación si Kaspersky Internet Security está instalado en el otro dispositivo. Para crear un comando SMS, tendrá que saber el código secreto de Kaspersky Internet Security instalado en su dispositivo.

Puede permitir o bloquear el inicio remoto de las funciones Antirrobo en su dispositivo. Resulta imposible iniciar Antirrobo de forma remota si su ejecución está bloqueada.

El acceso a la configuración de Antirrobo está protegido con un código secreto (se especifica durante la configuración inicial de Antirrobo o cuando ejecute la Protección de privacidad por primera vez).

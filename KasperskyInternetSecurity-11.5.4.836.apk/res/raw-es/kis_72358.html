Configuración inicial de Antirrobo

Las funciones de Antirrobo están desactivadas de forma predeterminada después de que la aplicación se haya instalado: no puede iniciarlas de forma remota mediante Gestión web o comandos SMS. Para activar las funciones de Antirrobo, tiene que realizar la configuración inicial de Antirrobo.

Después de la configuración inicial, todas las funciones de Antirrobo estarán activadas con la configuración recomendada por los especialistas de Kaspersky Lab.

El asistente para la configuración inicial de Antirrobo puede ejecutarse solamente una vez. Puede configurar más adelante la función Antirrobo en la configuración de la aplicación.

Para realizar la configuración inicial de Antirrobo:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Antirrobo. 

    De este modo, se iniciará el asistente para la configuración inicial de Antirrobo.

  2. Siga las instrucciones del asistente.

Para modificar la configuración de Antirrobo después de la configuración inicial:

En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Antirrobo.

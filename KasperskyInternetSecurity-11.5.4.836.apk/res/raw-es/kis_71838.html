Propósito del código secreto

Se le solicitará el código secreto en los casos siguientes:

  • Para acceder a Antirrobo y a la configuración de Protección de privacidad.
  • Al enviar un comando SMS desde otro dispositivo móvil para activar la alarma del dispositivo, bloquearlo, localizarlo en un mapa, borrar datos, u ocultar contactos confidenciales y la información relacionada de forma remota.

La aplicación le solicitará especificar el código secreto durante la configuración inicial de Antirrobo o cuando ejecute la Protección de privacidad por primera vez. Es posible modificar más tarde el código secreto instalado.

El código secreto está compuesto por cifras. Debe tener como mínimo cuatro caracteres.

Si olvida el código secreto, puede restablecerlo.

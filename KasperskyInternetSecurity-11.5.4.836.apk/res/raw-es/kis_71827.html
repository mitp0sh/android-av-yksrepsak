Acerca de Protección web y Antifraudes de mensajes de texto

Con Protección web y Antifraudes de mensajes de texto, puede visitar sitios web de confianza y utilizar datos personales en línea con seguridad.

Protección web y Antifraudes de mensajes de texto están disponibles únicamente en la versión Premium de Kaspersky Internet Security.

Antifraudes de mensajes de texto solo está disponible en dispositivos con tarjeta SIM insertada.

Protección web bloquea sitios web malintencionados que distribuyen código malicioso y sitios web fraudulentos que roban datos confidenciales y obtienen acceso a sus cuentas financieras.

Antifraudes de mensajes de texto bloquea vínculos en SMS que apunten a sitios web malintencionados o fraudulentos.

Protección web comprueba los sitios web antes de que se abran. Antifraudes de mensajes de texto también comprueba vínculos en mensajes SMS antes de que se abran. Protección web y Antifraudes de mensajes de texto realizan las comprobaciones mediante el servicio en la nube de Kaspersky Network Security (un servicio en línea de Kaspersky Lab dedicado exclusivamente a esta tarea que contiene información sobre la fiabilidad de archivos, aplicaciones y recursos web).

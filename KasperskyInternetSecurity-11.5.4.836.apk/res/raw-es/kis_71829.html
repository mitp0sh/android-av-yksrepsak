Análisis regular de sitios web

Para activar el análisis regular de sitios web:

  1. En el panel de inicio rápido de la ventana principal de Kaspersky Internet Security, toque Configuración > Protección web.
  2. Seleccione un valor para el Modo de protección Estándar o Ampliado.
  3. Toque Cambiar navegador (este botón aparecerá si Protección web está activado y si no se ha configurado el navegador estándar de Android como navegador predeterminado).

    Se iniciará el asistente de selección del navegador predeterminado.

  4. Siga las instrucciones del asistente.

    Cuando el asistente finalice, el navegador estándar de Android será el navegador predeterminado.

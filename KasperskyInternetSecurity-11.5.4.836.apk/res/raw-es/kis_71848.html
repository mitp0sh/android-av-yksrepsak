Bloquear y Localizar

Permite o impide el bloqueo y la localización del dispositivo a través de Gestión web de forma remota o mediante el comando SMS find: <código> (donde <código> es el código secreto para Kaspersky Internet Security en su dispositivo).

Los comandos SMS enviados se facturan de acuerdo con el plan de precios del otro dispositivo.

Si se permite esta función en el dispositivo, al recibir el comando Localizar y Bloquear la aplicación bloquea el dispositivo y muestra sus coordenadas en el sitio web. La aplicación devuelve un SMS con las coordenadas del dispositivo si envía un comando SMS. La aplicación también envía las coordenadas del dispositivo en un mensaje SMS al número de teléfono o dirección de correo electrónico que especifique en la sección Formas de recibir un nuevo número de teléfono y la ubicación del dispositivo de la configuración de Antirrobo.

Los mensajes SMS enviados se facturan de acuerdo con el plan de precios de su operador de servicios móviles.

Para enviar el correo electrónico, la aplicación envía un SMS al número especializado del operador móvil ruso MTS. El operador de servicios móviles MTS (Rusia) envía las coordenadas del dispositivo a su dirección de correo electrónico.

Los mensajes SMS enviados al número especial se cobran según la tarifa de su operador de servicios móviles.

Si la función está bloqueada en el dispositivo, la aplicación no podrá ejecutar el comando de Bloquear y Localizar.

La aplicación determina las coordenadas del dispositivo mediante GPS o redes inalámbricas y móviles.

Para permitir que la aplicación determine las coordenadas del dispositivo mediante GPS, debe activar el GPS en la configuración del dispositivo. Esto permite a la aplicación utilizar el módulo GPS cuando reciba el comando Bloquear y Localizar. Si la aplicación no logra obtener las coordenadas del dispositivo mediante GPS, utilizará redes inalámbricas y móviles para ello.

Navegadores compatibles

Protección web solo comprueba sitios web en el navegador estándar de Android y no en otros navegadores.

Cuando se active, el componente Protección web comprobará automáticamente cuál es el navegador predeterminado. Si Protección web no es compatible con el navegador predeterminado actual, este le invita a cambiar al navegador estándar de Android.

Para utilizar siempre Protección web al usar Internet, configure el navegador estándar de Android como navegador predeterminado.

Si no desea modificar el navegador predeterminado actual, inicie el navegador estándar de Android para utilizar sus datos personales en Internet de forma segura.

Puede iniciar el navegador estándar de Android desde Kaspersky Internet Security o desde el menú de aplicaciones del dispositivo. Si Protección web está activado, bloquea sitios web malintencionados y fraudulentos antes de que se abran.

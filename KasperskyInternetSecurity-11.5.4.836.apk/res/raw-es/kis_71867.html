Llamada entrante o mensaje de texto

Filtro de llamadas y SMS está disponible solo para dispositivos que dispongan de tarjeta SIM.

La aplicación le solicita elegir una acción que llevar a cabo cuando se recibe un mensaje SMS o se completa una llamada entrante.

La aplicación le solicita que seleccione una acción si se cumplen las siguientes condiciones: El modo Estándar está establecido en Filtro de llamadas y SMS y el número no se encuentra en la lista de contactos permitidos ni bloqueados.

Puede seleccionar una de las acciones siguientes:

  • Bloquear siempre.

    La aplicación marca las llamadas (o mensajes SMS) de los contactos como no deseados. La aplicación agrega el número o el contacto que emite la llamada (o mensaje SMS) a la lista de contactos bloqueados.

    Si se recibe una llamada del contacto, la aplicación bloquea todas las llamadas entrantes subsiguientes de este contacto. Sin embargo, si se recibe un mensaje SMS del contacto, la aplicación le solicita que seleccione la acción que se realizará con el mensaje entrante.

    Si se recibe un mensaje SMS del contacto, la aplicación bloquea ese mensaje SMS y los subsiguientes de dicho contacto. Sin embargo, si se recibe una llamada del contacto, la aplicación le solicita que seleccione la acción que se realizará con la llamada entrante.

  • Permitir siempre.

    La aplicación marca las llamadas (o mensajes SMS) del contacto como no deseados. La aplicación agrega el número o el contacto que emite la llamada (o mensaje SMS) a la lista de contactos permitidos.

    Si se recibe una llamada del contacto, la aplicación permite todas las llamadas entrantes subsiguientes de este contacto. Sin embargo, si se recibe un mensaje SMS del contacto, la aplicación le solicita que seleccione la acción que se realizará con el mensaje entrante.

    Si se recibe un mensaje SMS del contacto, la aplicación permite ese mensaje SMS y los subsiguientes de dicho contacto. Sin embargo, si se recibe una llamada del contacto, la aplicación le solicita que seleccione la acción que se realizará con la llamada entrante.

  • Omitir.

    La aplicación no lleva a cabo ninguna acción en la llamada (o SMS) de este contacto. La aplicación no agrega el número o el contacto que emite la llamada (o mensaje SMS) a ninguna de las listas.

    Cuando se reciba la siguiente llamada entrante (o mensaje SMS) del contacto, la aplicación le solicita que seleccione la acción que se realizará con dicho evento.

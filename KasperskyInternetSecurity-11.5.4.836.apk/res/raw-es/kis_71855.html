Envío de comandos SMS

Puede ejecutar funciones Antirrobo de manera remota u ocultar la información desde cualquier dispositivo con Kaspersky Internet Security instalado. Para ello, tiene que enviar un comando SMS especial.

El comando enviado es un SMS cifrado que contiene el código secreto de la aplicación en el terminal receptor. El usuario no tiene conocimiento de la recepción del comando SMS.

La aplicación en el otro dispositivo ejecuta el comando SMS si se cumple lo siguiente:

  • El dispositivo recibe cobertura móvil.
  • Se ha configurado Kaspersky Internet Security como administrador de dispositivos;
  • Se ha autorizado el inicio remoto de esta función en el dispositivo.

Los comandos SMS salientes son facturados al precio de su operador de servicios móviles.

Número del teléfono receptor del comando SMS

El número de teléfono del dispositivo al que se envía el comando SMS.

Código secreto para la aplicación en el dispositivo receptor del comando SMS

El código secreto para Kaspersky Internet Security en el dispositivo que recibe el comando SMS.

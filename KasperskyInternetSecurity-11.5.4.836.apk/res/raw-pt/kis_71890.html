Alterar o código secreto

O código secreto é definido quando a aplicação é executada pela primeira vez e é utilizado para:

  • Impedir o acesso não autorizado às definições da aplicação
  • activar o início remoto das funções de Anti-Roubo e Protecção de Privacidade a partir de outro dispositivo.

Pode alterar o código secreto.

Para alterar o código secreto:

  1. Na janela principal do Kaspersky Internet Security, toque em Configurações > Definições adicionais > Alterar código secreto.
  2. Introduza o código secreto actual no campo Introduza o código secreto actual e toque em Seg.
  3. Introduza o novo código secreto no campo Crie um novo código secreto e clique em Seg.
  4. Reintroduza o mesmo código no campo Reintroduzir o novo código e toque em Introduzir.

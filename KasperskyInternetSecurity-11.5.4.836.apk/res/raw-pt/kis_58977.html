Recuperação do código secreto

Para recuperar o código secreto da aplicação:

  1. Abra a Web Management em qualquer dispositivo.
  2. Inicie sessão em Web Management com a sua Conta Kaspersky utilizada durante a configuração inicial da aplicação.
  3. Seleccione o dispositivo cujo código secreto tem de ser recuperado.
  4. Prima o botão Recuperação do código secreto.

    O código de recuperação é apresentado em Web Management.

  5. Inicie o Kaspersky Internet Security no dispositivo.
  6. No painel de início rápido na janela principal do Kaspersky Internet Security, toque em Configurações > Anti-Roubo. 
  7. Quando lhe for solicitado que introduza o código secreto, toque em Menu > Recuperação do código secreto.
  8. Introduza o código de recuperação apresentado em Web Management.

    O código secreto é apresentado no ecrã do dispositivo.

  9. Introduza o código secreto no campo Introduza o código secreto restaurado.

A aplicação abre a janela Definições de Anti-Roubo.

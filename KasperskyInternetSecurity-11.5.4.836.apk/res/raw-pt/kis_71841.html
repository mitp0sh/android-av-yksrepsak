Iniciar funções de Anti-Roubo utilizando comandos de SMS

Pode iniciar as funções de Anti-Roubo no seu dispositivo utilizando comandos de SMS se um cartão SIM estiver inserido no seu dispositivo.

Se a Prot. Cartão SIM estiver activada no dispositivo extraviado, irá obter o novo número de telefone quando o seu cartão SIM for substituído no dispositivo. O número será enviado para o seu número de telefone adicional ou para o endereço de e-mail especificado nas definições de Anti-Roubo. Utilize o novo número de telefone para iniciar outras funções de Anti-Roubo no seu dispositivo.

Para iniciar as funções de Anti-Roubo utilizando um comando de SMS enviado de um dispositivo diferente, execute uma das operações seguintes:

  • Crie um comando noutro dispositivo e envie-o para o seu dispositivo a partir do Kaspersky Internet Security utilizando a função de comando de SMS (se a aplicação estiver instalada noutro dispositivo).

    Ao criar um comando de SMS, utilize o código secreto do Kaspersky Internet Security instalado no seu dispositivo.

  • Envie uma mensagem SMS para o seu dispositivo com o texto especial seguinte:
    • alarm: <código> – código para activar o alarme e bloquear o dispositivo (em que <código> corresponde ao código secreto do Kaspersky Internet Security no seu dispositivo);
    • find: <código> – para bloquear o dispositivo e obter as suas coordenadas;
    • wipe: <código> – para eliminar dados pessoais e dados do cartão de memória;
    • fullreset: <código> – para eliminar todos os dados do dispositivo e reverter o dispositivo para as predefinições

    Depois de todos os dados terem sido removidos e o dispositivo reposto com as predefinições, não será possível receber e executar comandos remotos.

Os comandos de SMS enviados são cobrados de acordo com o tarifário do operador móvel.

Enviar comando de SMS

Pode executar as função de Anti-Roubo remotamente ou ocultar informações confidenciais de qualquer dispositivo com o Kaspersky Internet Security instalado. Para tal, é necessário enviar um comando de SMS especial.

Um comando de SMS é enviado como uma SMS encriptada e contém o código secreto da aplicação definido noutro dispositivo. A recepção do comando de SMS não será notada.

A aplicação no outro dispositivo executa o comando de SMS se:

  • O dispositivo detecta um sinal de rede móvel;
  • O Kaspersky Internet Security tiver sido definido como o Administrador do Dispositivo;
  • o início remoto desta função é permitido no dispositivo.

Os comandos de SMS enviados são cobrados de acordo com o tarifário actual do operador móvel.

Número de telefone que recebe o comando de SMS

O número de telefone do dispositivo para onde o comando de SMS é enviado.

O código secreto da aplicação no dispositivo que recebe o comando de SMS

O código secreto do Kaspersky Internet Security no dispositivo que recebe o comando de SMS.

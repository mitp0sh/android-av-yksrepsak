Cloud Security Scan

Existe uma verificação adicional para aplicações recentemente instaladas, a partir do serviço cloud da Kaspersky Security Network (um serviço online especial da Kaspersky Lab que contém informações sobre a fiabilidade de ficheiros, aplicações e recursos da Internet). A verificação através do serviço cloud ajuda-o a proteger os dados de ameaças novas e desconhecidas.

Se a caixa de verificação Cloud Security Scan estiver seleccionada, o Kaspersky Internet Security irá verificar adicionalmente as aplicações recentemente instaladas antes de estas serem iniciadas pela primeira vez, utilizando o serviço cloud da Kaspersky Security Network. Se a caixa estiver desmarcada, o Kaspersky Internet Security não irá executar as verificações adicionais.

Esta definição está disponível se o modo de Protecção estiver definido como Recomendado ou Expandido.

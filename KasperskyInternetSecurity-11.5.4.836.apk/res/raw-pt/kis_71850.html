Limpeza de Dados

Utilize o site da Web o comando SMS wipe: <código> ou fullreset: <código> (em que <código> corresponde ao código secreto do Kaspersky Internet Security seu dispositivo) para permitir/bloquear a eliminação remota de dados pessoais ou de todos os dados do dispositivo.

Os comandos de SMS enviados são cobrados de acordo com o tarifário do operador móvel.

A Limpeza de Dados elimina os dados seguintes:

  • Apenas dados pessoais (por exemplo, contactos, mensagens e dados da conta Googleâ„¢) e dados do cartão de memória.
  • Todos os dados, incluindo os dados do cartão de memória. Adicionalmente, depois de todos os dados terem sido eliminados, o dispositivo é reposto com as predefinições de fábrica.

Depois de todos os dados terem sido removidos e o dispositivo reposto com as predefinições, não será possível receber e executar comandos remotos.

Se esta função estiver activada no dispositivo, a aplicação elimina dados pessoais ao receber o comando de limpeza de dados. Se enviar um comando de SMS, a aplicação devolve uma mensagem SMS a confirmar a eliminação dos dados.

As mensagens SMS enviadas são cobradas de acordo com o tarifário do operador móvel.

Se a função estiver bloqueada no dispositivo, não é possível a aplicação executar o comando de Limpeza de Dados.

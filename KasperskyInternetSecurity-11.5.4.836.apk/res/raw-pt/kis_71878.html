Ocultação de dados automática

A Protecção de Privacidade está disponível apenas com a versão Premium do Kaspersky Internet Security em dispositivos com um cartão SIM inserido.

Para gerir a Protecção de Privacidade (ver mensagens de texto e chamadas dos contactos ocultos, configurar definições), é necessário desactivar a ocultação de dados. Quando terminar a gestão da Protecção de Privacidade, tem de activar a ocultação de dados novamente. Pode configurar a activação automática da ocultação de dados após um determinado período, depois de terminar a gestão da Protecção de Privacidade (após o dispositivo mudar para o modo de poupança de energia).

Para activar o início automático da Protecção de Privacidade:

  1. No painel de início rápido na janela principal do Kaspersky Internet Security, toque em Configurações > Protecção de Privacidade. 
  2. Introduza o código secreto.

    Caso se esqueça do código secreto, pode recuperá-lo.

  3. Na opção Protecção de Privacidade seleccione desligado.
  4. Toque em Configurações.
  5. Seleccione o período após o qual a ocultação de dados automática deve ser activada. Para tal, seleccione um valor para a definição Ocultação automática.

Quando a ocultação de dados está desactivada, a aplicação activa a mesma automaticamente após um período especificado, depois de o dispositivo mudar para o modo de poupança de energia.

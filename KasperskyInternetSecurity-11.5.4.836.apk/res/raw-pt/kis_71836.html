Sobre o Anti-Roubo

O Anti-Roubo protege os seus dados contra acessos não autorizados e ajuda a localizar o dispositivo em caso de furto ou extravio.

O componente Anti-Roubo permite efectuar as operações seguintes remotamente:

  • Bloquear o dispositivo e determinar a sua localização
  • Activar um alarme sonoro no dispositivo
  • Eliminar dados do dispositivo
  • Tirar uma fotografia de identificação da pessoa que está actualmente a utilizar o dispositivo

Se o cartão SIM for substituído ou se o dispositivo for ligado sem um cartão SIM, pode bloquear o dispositivo remotamente e obter o novo número de telefone. Desta forma, é possível iniciar outras funções de Anti-Roubo no dispositivo extraviado.

A Prot. Cartão SIM está disponível apenas em dispositivos com um cartão SIM inserido.

Pode iniciar as funções de Anti-Roubo remotamente a partir de qualquer dispositivo através do site ou utilizando comandos SMS especiais. Pode enviar uma mensagem SMS padrão com um código especial a partir de outro dispositivo móvel. Se o Kaspersky Internet Security estiver instalado no outro dispositivo, pode enviar um comando de SMS a partir da aplicação. Para enviar um comando de SMS, é necessário o código secreto do Kaspersky Internet Security no seu dispositivo.

É possível permitir ou bloquear a execução remota das funções de Anti-Roubo no dispositivo. Se o início do componente Anti-Roubo estiver bloqueado, não é possível iniciar o mesmo remotamente.

O acesso às definições de Anti-Roubo está protegido com um código secreto (especificado durante a configuração inicial do Anti-Roubo ou quando a Protecção de Privacidade é iniciada pela primeira vez).

Definições de Limpeza de Dados

Para configurar as definições de Limpeza de Dados:

  1. No painel de início rápido na janela principal do Kaspersky Internet Security, toque em Configurações > Anti-Roubo.
  2. Introduza o código secreto.

    Caso se esqueça do código secreto, pode recuperá-lo.

  3. Seleccione Comandos permitidos.
  4. Marque ou desmarque a caixa de selecção Limpeza de Dados para permitir ou bloquear a detecção remota de informações pessoais (por exemplo, contactos, mensagens SMS, dados da agenda e dados do cartão de memória) bem como dados, incluindo os dados do cartão de memória.
